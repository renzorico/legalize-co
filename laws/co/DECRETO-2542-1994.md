# DECRETO 2542 DE 1994

(noviembre 17)

### por el cual se adiciona la escala de remuneración y la denominación y grado de los empleos de la Contraloría General de la República

**Subtipo:** DECRETO ORDINARIO

El Presidente de la República de Colombia, en desarrollo de las normas generales señaladas en la Ley 4º de 1992,

### DECRETA:

Artículo 1º. Adiciónase la escala de remuneración de los empleos del nivel Directivo - Asesor de la Contraloría General de la República, así:

| | |
| --- | --- |
| **ESCALA NIVEL DIRECTIVO - ASESOR** | |
| | |
| **Grado** | **Asignación básica** |
| 12 | $ 540.000 |
| 13 | 600.000 |
| 14 | 680.000 |
| 15 | 790.000 |
| 16 | 980.000 |
| 17 | 1.210.000 |

TEXTO CORRESPONDIENTE A []

Artículo 2º. Adiciónase la denominación y grado de remuneración de los empleos de Contraloría General de la República, establecido en la Ley 106 de 1993, así:

| | |
| --- | --- |
| **DENOMINACION** | **GRADO** |
| **NIVEL DIRECTIVO -ASESOR** | |
| Asesor | 19 |
| | 17 |
| | 16 |
| | 15 |
| | 14 |
| | 13 |
| | 12 |

Artículo 3º. El presente Decreto rige a partir de la fecha de su publicación y modifica las disposiciones que le sean contrarias.

Publíquese y cúmplase.

Dado en Santafé de Bogotá, D.C., a 17 de noviembre de 1994.
