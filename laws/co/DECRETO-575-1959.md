# DECRETO 575 DE 1959

(febrero 27)

### Por el cual se suprimen unos cargos y se crea otros

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

***Artículo 1°*** Suprímense los siguientes cargos de la Dirección General de Aduanas:

*Departamento Administrativo.*

*Servicios Especiales.*

| | | |
| --- | --- | --- |
| 2 Almacenistas, cada uno | $ | 450.00 |
| *Departamento de Resguardo.* | | |
| 1 Almacenista | | 450.00 |
| 1 Auxiliar | | 400.00 |

***Artículo 2°*** Créase la Proveeduría del Departamento Administrativo de la Dirección General de Aduanas, con los siguientes cargos:

| | | |
| --- | --- | --- |
| 1 Almacenista Jefe | $ | 700.00 |
| 1 Subalmacenista-Contador | | 600.00 |
| 1 Auxiliar de Kárdex | | 450.00 |

***Artículo 3°*** Las atribuciones y obligaciones del Almacenista Jefe de la Proveeduría de la Dirección General de Aduanas, serán las mismas que corresponden al Proveedor del Ministerio de Hacienda y Crédito Público.

***Artículo 4°*** El presente Decreto entrará a regir a partir del 1° de enero de 1959, y modifica el Decreto 2636 de 1958.

Afecta la vigencia de: []

Comuníquese y cúmplase.

Dado en Bogotá a 27 de febrero de 1959.

**ALBERTO LLERAS**.

El Ministro de Hacienda y Crédito Público,

## Hernando Agudelo Villa.
