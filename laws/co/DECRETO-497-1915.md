# DECRETO 497 DE 1915

(marzo 13)

### Por el cual se organiza la Administración Principal de Correos de Cali

**Subtipo:** DECRETO ORDINARIO

### *El Presidente de la República de Colombia*****.

en ejercicio de sus atribuciones legales,

### decreta:

**Artículo único** Organízase la Administración Principal de Correos de Cali, con el siguiente personal y asignaciones mensuales:

| | | |
| --- | --- | --- |
| *Sección* 1.ª | | |
| Un Administrador Principal, con | $ | 100 |
| Un Contador, con | | 50 |
| Un Ayudante Contador, con | | 30 |
| Un Copista Escribiente, con | | 30 |
| *Sección 2.ª (Encomiendas).* | | |
| Un Jefe, con | | 45 |
| Un Ayudante, con | | 25 |
| *Sección 3.ª (Oficina de Recibo).* | | |
| Un Jefe, con | | 40 |
| Un Expendedor, de Especies Postales con | | 35 |
| Un Oficial de Apartados, con | | 20 |
| Un Portero Escribiente, con | | 16 |
| *Sección 4.ª (Correo Urbano).* | | |
| Un Jefe con | | 25 |
| Dos Buzoneros, cada uno con | | 14 |
| Tres Carteros, cada uno, con | | 14 |

Quedan suprimidas una de las plazas de Conductor de valijas para el servicio del correo diario entre Cali y Buenaventura y dos de Cartero en aquella Oficina.

Comuníquese y publíquese

Dado en Bogotá a 13 de marzo de 1915.

### JOSE VICENTE CONCHA

El Ministro De Gobierno,

**miguel** ABADIA MENDEZ
