# DECRETO 1243 DE 1938

(julio 13)

### Por el cual se crean unas Oficinas Postales y Telegráficas y se proveen unas plazas creadas por los Decretos números 944 y 1148 de 1938

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

**Artículo 1°** Créanse, las siguientes Oficinas Postales y Telegráficas:

| | | |
| --- | --- | --- |
| **Atlántico.** | | |
| 2026-Malambo. | | |
| Un Telegrafista Administrador de Córreos con | $ | 40.00 |
| **Bolívar.** | | |
| 3087--Morroa. | | |
| Un Telegrafista Administrador de Correos con | $ | 40.00 |

**Parágrafo**. El gasto que demande el cumplímiento del presente artículo, se imputará al Capítulo 56, artículo 385 del Presupueste vigente.

**Artículo 2°** Provéese la siguiente plaza creada por el Decreto número 944 de 25 de mayo de 1938:

Manuel J. Vega G., Telegrafista Administrador de Correos de Repelón (Atlántico).

**Artículo 3°** Provéense las siguientes plazás creadas por el Decreto número 1140 de junio 39 de 1938:

Luis Eduardo Montoya, Cartero de l a Oficina Mixta de Concepción (Antioquia).

Juana Peña, Cartero de la Oficina Mixta de San Roque (Antioquia).

Emilio Montes, Cartero de la Oficina Telegráfica de Sevilla (V).

Juan de Dios Ramírez, Cartero de la Oficina Mixta de Carmen (Chocó).

Comuníquese y publíquese.

Dado en La Mesa, a 13 de julio de 1938.

### ALFONSO LOPEZ

Ministro de Guerra, encargado del Despacho de Correos y Telégrafos,

### Alberto PUMAREJO
