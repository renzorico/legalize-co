# DECRETO 974 DE 1903

(octubre 30)

### Por el cual se reorganiza una Oficina Postal, se aumentan las asignaciones de otras y se crea una

**Subtipo:** DECRETO ORDINARIO

## El Vicepresidente de la República, encargado del Poder Ejecutivo,

En uso de sus facultades legales,

### DECRETA:

**Art. 1.°** Desde el día 1° de Noviembre próximo venidero en adelante reorganízase la Administración de Correos nacionales de Bucaramanga, con el siguiente personal y asignaciones mensuales:

| | | |
| --- | --- | --- |
| Un Administrador | $ | 2,000 |
| Un Tenedor de Libros | | 1,200 |
| Un Oficial de Correspondencia | | 1,200 |
| Un íd de Encomiendas | | 1,200 |
| Un Portero | | 700 |

**Art. 2.°** Auméntese el sueldo de los empleados de las Administraciones de Correos nacionales de las poblaciones que se Expresan, desde la misma fecha citada, en la siguiente:

| | | |
| --- | --- | --- |
| *Cartago* | | |
| El de Administrador, á | $ | 1,200 |
| El de Ayudante, á | | 700 |
| *Pamplona* | | |
| El Administrador, á | | 1,000 |

**Art. 3.°** Créase en Guapí (Departamento del Cauca) Oficina de Correos nacionales, con el personal y asignaciones por mes, que se expresan, en plata:

| | | |
| --- | --- | --- |
| Un Administrador de Correos, con | $ | 60 |
| Para gastos de útiles de escritorio | | 5 |

Comuníquese y publíquese.

Dado en Bogotá, á 30 de Octubre de 1903.

### JOSE MANUEL MARROQUÍN

El Ministro de Gobierno,

### ESTEBAN JARAMILLO
