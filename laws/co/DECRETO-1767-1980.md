# DECRETO 1767 DE 1980

(julio 11)

### Por el cual se dicta una disposición en la Planta Interna del Ministerio de Relaciones Exteriores

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades constitucionales,

### DECRETA:

**Artículo 1°** Suprímese en la Planta Interna de Personal del Ministerio de Relaciones Exteriores, el siguiente cargo:

| | | | |
| --- | --- | --- | --- |
| Denominación del empleo | N° cargos | Código | Grado |
| Oficina de Planeación. | | | |
| Profesional Universitario | 1 | 3020 | 6 |

**Artículo 2°** Créase en la Planta Interna de Personal del Ministerio de Relaciones Exteriores, el siguiente cargo:

| | | | |
| --- | --- | --- | --- |
| Denominación del empleo | N° cargos | Código | Grado |
| Oficina de Planeación. | | | |
| Asesor | 1 | 1020 | 2 |

**Artículo 3°** Nombrase al doctor Guillermo Triana Ayala, Asesor, código 1020, grado 02 de la Oficina de Planeación, cargo creado, el artículo 2 del presente Decreto.

**Artículo 4°** El presente Decreto rige desde su expedición y modifica la Planta Interna del Personal del Ministerio de Relaciones Exteriores, establecida mediante Decreto 1371 de 1980.

Comuníquese y publíquese.

Dado en Bogotá, D. E., a 11 de julio de 1980.

### JULIO CESAR TURBAY AYALA

El Ministro de Relaciones Exteriores, encargado, **Julio Londoño Paredes.** El Ministro de Hacienda y Crédito Público, **Jaime García Parra.** El Jefe del Departamento Administrativo del Servicio Civil **Laura Ochoa de Ardila.**
