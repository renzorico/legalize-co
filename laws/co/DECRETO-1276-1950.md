# DECRETO 1276 DE 1950

(abril 20)

### Por el cual se distribuye una partida

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia

en uso de sus facultades legales,

### decreta :

**Artículo primero**. De los fondos apropiados en él artículo 1165, Capítulo 94 del Presupuesto Nacional vigente, "para atender al sostenimiento de becas de diez estudiantes ecuatorianos en el año", se distribuyen así:

| | | | |
| --- | --- | --- | --- |
| Capítulo 94, articulo 1165. | | | |
| | | *Mensual.* | *Total.* |
| Para atender al pago de 10 pensiones de estudiantes ecuatorianos en la Universidad Nacional, a razón de $ 150 mensuales, enero a diciembre, inclusive | $ | 1.500.00 | 18:000.00 |
| Para gastos de matrícula, compra de libros y-gastos de instalación de los mismos estudiantes, a razón de $ 200 cada uno | | | 2.000.00 |
| Total | $ | | 20.000.00 |

**Artículo segundo**. Tanto las pensiones de los becados como la partida señalada para gastos de matrícula, compra de libros e instalación de los mismos, se pagarán por conducto del Síndico de. la Universidad Nacional.

Comuníquese publíquese y cúmplase.

Dado en Bogotá a 20 de abril de 1950.

## MARIANO OSPINA PEREZ

El Ministrode Hacienda y Crédito Público, ***Hernán******JARAMILLO OCAMPO***-El Ministro de Educación Nacional, **Manuel MOSQUERA GARCES.**
