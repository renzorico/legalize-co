# DECRETO 2056 DE 1943

(octubre 13)

### Por el cual se hacen unas modificaciones en el personal del ramo de Correos del Ministerio de Correos y Telégrafos

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en uso de sus facultades legales,

DECRETA:

**Articulo 1°** A partir del 1° de noviembre próximo, suprimense las siguientes plazas:

*Departamento de Correos.*

*Sección de Pasaportes e Itinerarios:*

| | |
| --- | --- |
| La plaza de Mecanógrafa 2a, con asignación mensual de | 70.00 |
| que desempeña Olga M. Zapata. | |
| *Pauna* (Boyacá): La plaza de Agente de Correos, con asignación mensual de | 15.00 |
| que desempeña María Quiroga de López. | |
| Suma | 85.00 |

| | |
| --- | --- |
| **Artículo 2°** A partir del 19 de noviembre próximo créanse las siguientes plazas: | |
| *Administración de Correos de Chapinero:* | |
| Un Auxiliar 2°, con asignación mensual de | 55.00 |
| y se nombra para desempeñar esta plaza a Fanny Olarte | |
| Arango. | |
| *El Cairo* (Valle): | |
| Un Agente de Correos, con asignación mensual de | 30.00 |
| y se nombra para desempeñar esta plaza a Julio Jaramillo. | |
| Suma | 85.00 |

**Artículo 3°** El gasto que demande el cumplimiento, del presente Decreto se imputará al capítulo 74, articulo 1107, del Presupuesto vigente.

Comuníquese y publíquese.

Dado en Bogotá a 13 de octubre de 1943.
