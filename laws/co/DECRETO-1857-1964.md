# DECRETO 1857 DE 1964

(julio 22)

### Por el cual se distribuye una partida

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus atribuciones legales, y

CONSIDERANDO:

que por Decreto número 1496 de 23 de junio del año en curso, por el cual se hacen unas traslaciones en el Presupuesto de Gastos para la vigencia fiscal de 1964 (Ministerio de Educación Nacional), se destinó la partida de $ 85.000.00, para gastos del Museo "Jorge Eliécer Gaitán", suma que es necesario distribuir,

### DECRETA:

**Artículo primero.** La partida apropiada por el Decreto *número* 1496 de junio 23 del presente año, para el Capítulo 704, artículo 7324, del Presupuesto de la actual vigencia, se distribuye así:

| | | |
| --- | --- | --- |
| a) Sueldos | $ | 400.00 |
| c) Servicios públicos | | 597.00 |
| d) Materiales y suministros | | 9.839.71 |
| f) Mantenimiento y aseguros | | 69.673.39 |
| h) Jornales | | 1.520.00 |
| i) Impresos y publicaciones | | 2.970.00 |
| Total | $ | 85.000.00 |

Comuníquese y cúmplase.

Dado en Bogotá, D. E., a 22 de julio de 1964.
