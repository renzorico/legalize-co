# DECRETO 522 DE 1911

(junio 03)

### por el cual se abre un crédito suplemental al Presupuesto nacional de gastos del año económico de 1911, imputable á los departamentos de Relaciones Exteriores é Instrucción Pública.

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en ejercicio de sus facultades legales, y

### CONSIDERANDO

que el Consejo de Ministros, en la sesión del 31 de Mayo último, á solicitud de los Ministerios de Relaciones Exteriores é Instrucción Pública, resolvió la apertura de un crédito suplemental al Presupuesto de gastos de la vigencia fiscal en curso, y en vista de los expedientes formados con tal objeto en los citados Ministerios,

### DECRETA:

**Artículo 1.°** Abrese al Presupuesto nacional de gastos del año económico de 1911 un crédito suplemental por la suma de treinta y seis mil ochocientos pesos ($36,800), con la siguiente imputación:

| | | | |
| --- | --- | --- | --- |
| MINISTERIO DE RELACIONES EXTERIORES | | | |
| departamento de relaciones exteriores | | | |
| *Capítulo* 26. | | | |
| Servicio diplomático. | | | |
| Artículo 176. Para gastos de servicio diplomático, en once meses | $ | 30.000 | 30.000 |
| MINISTERIO DE INSTRUCCIÓN PUBLICA | | | |
| departamento de instrucción publica | | | |
| *Capítulo* 55. | | | |
| Gastos varios. | | | |
| Artículo 318. Para gastos extraordinarios é imprevistos | $ | 6.800 | 6.800 |
| Total | | $ | 36.800 |

**Artículo 2.°** Por los Ministerios de Relaciones Exteriores é Instrucción Pública se dará oportuno cumplimiento á lo dispuesto en el artículo 6.° de la Ley 19 de 1894.

**Parágrafo.** Las oficinas ordenadoras y pagadoras respectivas describirán en sus libros las operaciones á que da lugar el presente Decreto.

Publíquese.

Dado en Bogotá á 3 de Junio de 1911.

### CARLOS E. RESTREPO

El Ministro del Tesoro,

**g**. MARTINEZ A.
