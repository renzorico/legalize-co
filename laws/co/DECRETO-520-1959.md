# DECRETO 520 DE 1959

(febrero 24)

### Por el cual se suprimen unos cargos y se crean otros en la Superintendencia Nacional de Importaciones

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

En uso de sus atribuciones legales,

### DECRETA:

**Artículo 1°** A partir del 1° de marzo del año en curso, suprímense los siguientes cargos en la Superintendencia Nacional de Importaciones:

| --- | --- | --- | --- |
| 1 | Chofer | $ | 300.00 |
| 1 | Mecanógrafa de 2ª | | 450.00 |

| 1 | Jefe de Listas | | 1.500.00 |
| 1 | Estadígrafo | | 1.200.00 |

| | | | |

| 1 | Mecanógrafa Recepcionista | | 400.00 |
| 1 | Economista | | 1.100.00 |

| 1 | Clasificador 2° de Registros | | 600.00 |
