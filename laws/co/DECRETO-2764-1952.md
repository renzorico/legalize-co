# DECRETO 2764 DE 1952

(noviembre 04)

### Por el cual se suprimen unos cargos, se crean otros y se modifican unas asignaciones en el Ministerio de Educación Nacional

**Subtipo:** DECRETO LEGISLATIVO

### El Designado, encargado de la Presidencia de la República de Colombia,

en uso de sus facultades legales y en especial de las que le confiere el Decreto legislativo número 289 de 1950,

### DECRETA:

**Artículo primero.** A partir del primero de noviembre del presente año, el personal y asignaciones del equipo de Calzado Escolar, dependiente del Ministerio de Educación Nacional, serán los siguientes:

| | | |
| --- | --- | --- |
| Director | $ | 500.00 |
| Encargado del Equipo y Almacenes | | 380.00 |
| Contador-ayudante | | 200.00 |
| Almacenista-vendedor | | 250.00 |
| Ayudante-vendedor | | 200.00 |
| Mecanógrafa | | 220.00 |
| Mecánico | | 230.00 |
| Oficial-inspector | | 225.00 |

**Artículo segundo**. El encargado del Equipo y Almacenes será el responsable del manejo de los elementos del Calzado Escolar y en consecuencia constituirá la fianza, de acuerdo con el artículo 13 del Decreto legislativo número 911 de 1932.

**Artículo tercero.** Suprímense los siguientes cargos en el Departamento de Contabilidad, Control y Administración del Ministerio de Educación Nacional:

| | | |
| --- | --- | --- |
| Ayudante Contabilista | $ | 400.00 |
| Chofer | | 190.00 |

**Artículo cuarto.** Con base en las supresiones de que trata el artícuo anterior, créanse los siguientes Cargos en el Departamento de Contabilidad, Control y Administración del Ministerio de Educación Nacional:

| | | |
| --- | --- | --- |
| Mecanógrafa-oficial de Contabilidad | $ | 300.00 |
| Oficial de Control de Reservas y Pedidos | | 250.00 |

**Artículo quinto.** A partir del primero de noviembre los sueldos del Ayudante Segundo de la Pagaduría Principal y el de la Mecanógrafa-Ayudante del Pagador de Pensiones, serán de $ 340.00 y $ 265.00 respectivamente.

Comuníquese y publíquese.

Dado en Bogotá a 4 de noviembre de 1952.

### ROBERTO URDANETA ARBELAEZ

El Ministro de Hacienda y Crédito Público,

## Antonio Alvarez Restrepo

El Ministro de Educación Nacional,
