# DECRETO 3070 DE 1989

(diciembre 29)

### Por el cual se suprimen y se crean unos cargos en la Planta Interna de Personal del Ministerio de Relaciones Exteriores

**Subtipo:** DECRETO ORDINARIO

El Presidente de la República de Colombia en uso de las facultades que le confiere el numeral 21 del artículo 120 de la Constitución Política,

### DECRETA:

**Artículo 1°.** Suprimense en la Planta Interna de Personal del Ministerio de Relaciones Exteriores, los siguientes cargos:

| | | | |
| --- | --- | --- | --- |
| **N°DE CARGOS** | **DEPENDENCIA Y DENOMINACION** | **CODIGO** | **GRADO** |
| | SUBSECRETARIA DE ORGANISMOS Y CONFERENCIAS INTERNACIONALES | | |
| 1 | Profesional Especializado | 3010 | 10 |
| | SUBSECRETARIA DE ASUNTOS ADMINISTRATIVOS | | |
| | SECCION DE REGISTRO Y DESPACHO DE CORRESPONDENCIA | | |
| 1 | Profesional Universitario | 3020 | 06 |
| | DIVISION DE ASUNTOS JURIDICOS | | |
| | SECCION DE CONCEPTOS Y DICTAMENES | | |
| 1 | Profesional Universitario | 3020 | 03 |

**Artículo 2°.** Créanse en la Planta Interna de Personal del Ministerio de Relaciones Exteriores, los siguientes cargos:

| | | | |
| --- | --- | --- | --- |
| **N°DE CARGOS** | **DEPENDENCIA Y DENOMINACION** | **CODIGO** | **GRADO** |
| | DESPACHO DEL MINISTRO | | |
| 1 | Asesor | 1020 | 04 |
| 1 | Secretario | 5140 | 10 |
| 1 | Auxiliar Administrativo | 5120 | 09 |

**Artículo 3°.** Las erogaciones que ocasione el cumplimiento del presente Decreto se pagarán con cargo al presupuesto del Ministerio de Relaciones Exteriores.

**Artículo 4°.** Este Decreto rige a partir de la fecha de su publicación y modifica en lo pertinente los Decretos 1371 de 1980 y 2605 de 1983.

Afecta la vigencia de: []

Publíquese, comuníquese y cúmplase.

Dado en Bogotá, D. E., a 29 de diciembre de 1989.
