# DECRETO 1062 DE 1916

(junio 19)

### Por el cual se reorganizan dos oficinas en el ramo Telegráfico

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República,

en uso de sus facultades legales,

### decreta:

**Artículo 1.°** A partir del primero de julio próximo, reorganízanse las Oficinas Postales y Telegráficas de Caldas y Cisneros (Departamento del Valle), cada una con el personal y asignaciones mensuales siguientes:

| | | |
| --- | --- | --- |
| **Caldas.** | | |
| Un Telegrafista Administrador de Correos, con | $ | 65 |
| Un Cartero | | 6 |
| Para gastos de alumbrado | | 1 |
| **Cisneros.** | | |
| Un Telegrafista Administrador de Correos, con | $ | 70 |
| Un Ayudante | | 60 |
| Un Cartero | | 4 |
| Para gastos de alumbrado | | 3 |

**Artículo 2.°** Los empleados de la Oficina de Caldas tendrán derecho a continuar disfrutando del aumento del 20 por 100 que les señala el Decreto número 57, de 13 de enero del corriente año, mientras dure el mal estado sanitario de aquel lugar.

Comuníquese y publíquese.

Dado en Bogotá a 19 de junio de 1916.

### JOSE VICENTE CONCHA**-El Ministro De Gobierno, **Miguel ABADIA MENDEZ.
