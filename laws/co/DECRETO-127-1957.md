# DECRETO 127 DE 1957

(julio 16)

### Por el cual se abre un crédito adicional - extraordinario- al Presupuesto vigente (Ministerio de Obras Públicas)

**Subtipo:** DECRETO LEGISLATIVO

### La Junta Militar de Gobierno de la República de Colombia,

Obrando de conformidad con las disposiciones del artículo 121 de la Constitución Nacional.

**DECRETA**:

***Artículo primero.*** Adiciónanse los cómputos líquidos del Presupuesto de Rentas e Ingresos del Tesoro de la Nación para la presente vigencia fiscal, con la cantidad de ciento veintiocho mil pesos ($ 128.000.00) moneda corriente, que se incorpora con imputación al siguiente numeral:

| | | |
| --- | --- | --- |
| RENTAS DE IMPOSICION | | |
| *c) Tasas y Multas.* | | |
| Numeral 69. Producto del Puerto Terminal Marítimo de Tumaco | $ | 128.000.00 |
| Total | $ | 128.000.00 |

| | | |
| --- | --- | --- |
| ***Artículo segundo.*** Con base en el nuevo recurso fiscal de que trata el artículo anterior, ábrese el siguiente crédito adicional --extraordinario-- al Presupuesto de Gastos para la vigencia fiscal en curso: | | |
| MINISTERIO DE OBRAS PUBLICAS | | |
| CAPITULO 530 | | |
| *Terminales Marítimos y Fluviales.* | | |
| *Terminal Marítimo de Tumaco.* | | |
| 001. 003. 206. 306. Al artículo 5354-A. Para gastos de administración y explotación del terminal Marítimo de Tumaco (Decreto 398 de 1957), en el segundo semestre del presente año | $ | 128.000.00 |
| Total | $ | 128.000.00 |

***Artículo tercero.*** El presente Decreto rige desde la fecha de su expedición y suspende todas las disposiciones que le sean contrarias.

Comuníquese, publíquese y cúmplase.

Dado en Bogotá, D. E., a 16 de julio de 1957.
