# DECRETO 792 DE 1915

(mayo 05)

### Por el cual se reorganiza la Sección de Policía Nacional de San Andrés y Providencia

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia

### DECRETA:

**Artículo único.** Desde el 1.° de junio próximo la Sección de San Andrés y Providencia, perteneciente a la séptima División de la Policía Nacional, constará del siguiente personal, con las asignaciones mensuales que se expresan:

| | | |
| --- | --- | --- |
| Un Comisario Jefe cien pesos | $ | 100 |
| Un Comisario de segunda clase ochenta pesos | | 80 |
| Un Comisario Mayor de tercera' clase sesenta pesos | | 60 |
| Un Comisario de tercera clase cincuenta pesos | | 50 |
| Un Secretario cuarenta pesos | | 40 |
| Cuatro Agentes de primera clase a $ 35 cada uno | | 140 |
| Ocho Agentes de segunda clase a 32 | | 256 |
| Ochenta y ocho Agentes de tercera clase a $ 30 cada uno. | | 2.640 |

Comuníquese y publíquese.

Dado en Bogotá a 5 de mayo de 1915.

### JOSE VICENTE CONCHA

El Ministro de Gobierno,

**MIGUEL** ABADIA MENDEZ
