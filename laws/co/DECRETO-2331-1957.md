# DECRETO 2331 DE 1957

(noviembre 20)

### Por el cual se fija el personal y asignaciones de los Juzgados de Menores de Bogotá

**Subtipo:** DECRETO ORDINARIO

### La Junta Militar de Gobierno de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

***Artículo primero*.** A partir del primero (1º) de noviembre del año en curso, el personal y asignaciones mensuales de los Juzgados de Menores de Bogotá, será como sigue:

| | | |
| --- | --- | --- |
| JUZGADO CIVIL DE MENORES | | |
| 1 Juez | $ | 1.100.00 |
| 1 Secretario | | 460.00 |
| 1 Oficial Mayor | | 345.00 |
| 2 Delegados de Estudio, cada uno | | 345.00 |
| 1 Portero Vigilante | | 230.00 |
| 1 Oficial de Contabilidad | | 345.00 |
| JUZGADO DE PROTECCION Y REHABILITACION DE MENORES | | |
| 1 Juez a | $ | 1.100.00 |
| 1 Secretario | | 460.00 |
| 1 Oficial Mayor | | 345.00 |
| 2 Visitadores Sociales, cada uno | | 345.00 |
| 1 Portero Vigilante | | 230.00 |

***Artículo segundo.*** Asimismo el Ministerio de Justicia nombrará para el servicio conjunto de los dos Juzgados, el siguiente personal, al que corresponderá las asignaciones mensuales que en cada caso se indican:

| | | |
| --- | --- | --- |
| 1 Promotor Curador | $ | 1.915.00 |
| 1 Auxiliar del Promotor | | 300.00 |
| 1 Médico Psiquiatra | | 550.00 |
| 2 Choferes cada uno | | 253.00 |
| 1 Conserje | | 172,50 |

***Artículo tercero*****.** Este Decreto rige a partir del primero (1º) de noviembre del año en curso.

Comuníquese y cúmplase.

Dado en Bogotá a 20 de noviembre de 1957.

Mayor General **Gabriel París G.,**

Presidente de la Junta.
