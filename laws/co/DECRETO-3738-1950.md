# DECRETO 3738 DE 1950

(diciembre 20)

### Tarifas en el servicio de apartados

**Subtipo:** DECRETO ORDINARIO

en uso de sus facultades legales,

DECRETA:

**Artículo único.** A partir del 1° de enero de 1951, regirán las siguientes tarifas en el servicio de apartados:

| | |
| --- | --- |
| Derechos de. apartado clase A., por año | $ 20.00 |
| Derechos da apartado clase B., por año | 15 00 |
| Derechos de apartado clase C., por año | 10.00 |
| Derechos de apartado clase D., por año | 5.00 |
| Depósito de garantía al tomar al apartado | 2.00 |
| Esta categoría de apartado corresponde a los instalados en Bogotá. | |
| Los derechos de apartado para el resto de oficinas del país serán los siguientes: | |
| Derechos de apartado pequeño, por año | 10.00 |
| Derechos de apartado mediano, por año | 15.00 |
| Derechos de apartador grande, por año | 20.00 |
| Derechos de garantía al tomar el apartado | 2.00 |

| | |
| --- | --- |
| Comuníquese y publíquese. Dado en Bogotá a 20 de diciembre de 1950. ***LAUREANO GOMEZ*** El Ministro de Correos y Telégrafos, ***José Tomás ANGULO*** | |
