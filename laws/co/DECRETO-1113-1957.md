# DECRETO 1113 DE 1957

(junio 05)

### Por el cual se distribuye la partida apropiada en el Presupuesto Nacional vigente, para dotación de los establecimientos de educación, dependientes de la División de Educación Femenina

**Subtipo:** DECRETO ORDINARIO

### La Junta Militar de Gobierno,

### DECRETA:

**Artículo primero.** La partida de treinta mil pesos ($ 30.000.00) destinada en el Capitulo 345, artículo 3518 del Presupuesto Nacional vigente, para dotación de los establecimientos de educación dependientes de la División de Bachillerato, se distribuye así:

Capítulo 346, artículo 3518:

| | | |
| --- | --- | --- |
| Ordinal 1º Colegio Nacional "Santa Librada", de Neiva | $ | 5.950.00 |
| Ordinal 2º Colegio Nacional "Simón Bolívar", de Garzón | | 5.530.00 |
| Ordinal 3º Colegio Nacional de Bachillerato de El Banco | | 1.750.00 |
| Ordinal 4º Colegio "San José de Guanentá", de San Gil | | 2.050.00 |
| Ordinal 5º Colegio "Custodio García Rovira", de Málaga | | 2.600.00 |
| Ordinal 8º Colegio Nacional "Sergio Arboleda", de Bogotá. | | 7.100.00 |
| Ordinal 7º Colegio "San Luis Gonzága", de Túquerres | | 5.000.00 |
| Total | $ | 30.000.00 |

**Artículo segundo.** Las sumas destinadas por los ordinales 3º y 7º del artículo primero del presente Decreto, para dotación de los Colegios Nacionales de Bachillerato de El Banco (Magdalena), y Túquerres (Nariño), respectivamente, se girarán a los Habilitados de los respectivos planteles.

Comuníquese y publíquese.

Dado en Bogotá a 7 de junio de 1957.
