# DECRETO 2423 DE 1928

(diciembre 21)

### Por el cual se crean unos empleados supernumerarios en la Aduana de Barranquilla

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

**Artículo único.** Créanse los siguientes puestos de empleados supernumerarios para la Administración de la Aduana de Barranquilla, por el término de seis meses, contados desde el 1º de enero del año entrante, y con las asignaciones que se expresan:

| | | |
| --- | --- | --- |
| Un Escribiente para la Sección de Anotación, con | $ | 90 |
| Un Oficial Ayudante del Reconocedor de Encomiendas Postales Reconocedor de Encomiendas Postales del Exterior, con | $ | 90 |
| Un Ayudante del Registrador de sobordos de carga, con | | 80 |
| Un Registrador de memoriales y reclamos, con | | 90 |
| Un Ayudante Escribiente para la Sección Fluvial, con | | 90 |
| Tres Ayudantes Bodegueros, cada uno con | | 100 |

Cópiese, comuníquese y publíquese.

Dado en Bogotá a 21 de diciembre de 1928.

### MIGUEL ABADIA MENDEZ

El Ministro de Hacienda y Crédito Público,

### Esteban JARAMILLO
