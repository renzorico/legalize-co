# DECRETO 1193 DE 1935

(julio 03)

### Por el cual se crean varios puestos en el ramo de Guerra

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

**Artículo 1°** Créanse en el Comando del destacamento del Amazonas, Putumayo y Caquetá, con destino a la lancha *Nely*, los siguientes puestos, con las asignaciones mensuales que a continuación se expresan:

| | | |
| --- | --- | --- |
| Un Motorista | $ | 45 |
| Un Timonel | | 30 |
| Un Marinero | | 15 |

| | | |
| --- | --- | --- |
| Dos Jefes de taller, a $ 45 cada uno | $ | 90 |
| Cinco Rancheros, a $ 15 cada uno | | 75 |
| Cinco soldados a $ 450 cada uno | | 22 50 |

**Artículo 2°** Auméntase en los siguientes puestos la dotación de personal auxiliar del Batallón *Sucre* número 2 de Infantería, con las asignaciones mensuales que a continuación se expresan:

Comuníquese y publíquese.

Dado en Bogotá a 3 de julio de 1935.

## ALFONSO LOPEZ

El Ministro de Guerra,

### Marco A.AULI
