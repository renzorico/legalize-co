# DECRETO 1673 DE 1916

(septiembre 30)

### Por el cual se reforma el marcado con el número 1505 del corriente año

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la Republica,

en uso de sus facultades legales,

### decreta:

Reorganízase la Oficina Telegráfica de Armenia (Departamento de Caldas) con el personal y asignaciones mensuales siguientes:

| | | | |
| --- | --- | --- | --- |
| Un Telegrafista Jefe, con | | | $ 60 |
| Dos Ayudantes, cada uno con | | $ 40 | 80 |
| Un Oficial de Recibo | | | 25 |
| Un Cartero | | | 12 |

Queda en estos términos reformado el Decreto número 1505, de 31 de agosto último.

Comuníquese y publíquese.

Dado en Bogotá a 30 de septiembre de 1916.

### JOSE VICENTE CONCHA.**- El Ministro de Gobierno, **Miguel ABADIA MENDEZ.
