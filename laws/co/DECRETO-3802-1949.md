# DECRETO 3802 DE 1949

(noviembre 30)

### Por el cual se hacen unos traslados en el Presupuesto vigente

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus atribuciones legales, y

CONSIDERANDO

que el Consejo de Ministros, en su sesión del día 30 de noviembre del año en curso, autorizó al Ministerio de Hacienda y Crédito Público para efectuar unos traslados dentro de las apropiaciones de los Ministerios de Higiene y Obras Públicas, según consta en el acta respectiva,

### DECRETA:

**Artículo 1°** Hácense los siguientes traslados en el Presupuesto vigente:

| | | |
| --- | --- | --- |
| MINISTERIO DE OBRAS PUBLICAS | | |
| *Capítulo 117.* | | |
| Del artículo 2308. Para ampliación, adiciones y mejoras, gastos de conservación y reparación de los edificios de propiedad nacional, o en servicio oficial, de fuera de la capital de la República, en el año | $ | 1.974.00 |
| *Capítulo 121.* | | |
| Del artículo 2567. Para la terminación de la pavimentación de Santa Marta | | 34.000.00 |
| Suman los contracréditos | $ | 35.974.00 |
| MINISTERIO DE HIGIENE | | |
| *Capítulo 56.* | | |
| Al artículo 796-A. Para la terminación de la pavimentación y alcantarillado de Santa Marta | $ | 34.000.00 |
| MINISTERIO DE OBRAS PUBLICAS | | |
| *Capítulo 118.* | | |
| Al artículo 2373. Para continuar la construcción del Colegio Oficial Instituto "Isidro Parra", del Líbano | | 1.974.00 |
| Suman los créditos | $ | 35.974.00 |

**Articulo 2°** El presente Decreto rige desde su fecha.

Comuníquese y publíquese.

Dado en Bogotá a 30 de noviembre de 1949.
