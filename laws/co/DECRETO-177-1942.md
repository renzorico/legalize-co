# DECRETO 177 DE 1942

(enero 27)

### Por el cual se hacen unas modificaciones en el personal de Telégrafos del Ministerio de Correos y Telégrafos

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en uso de sus facultades legales,

DECRETA:

**Artículo 1°** Hácense las siguientes modificaciones en el personal de Telégrafos del Ministerio de Correos y Telégrafos:

Supresiones:

Se suprimen las siguientes plazas de Guardas:

| | | |
| --- | --- | --- |
| *Chiriguana:* | | |
| Guarda "Chiriguaná-Bocapotreronone (Zona 9), con | $ 28.00 | |
| *El Paso:* | | |
| Guarda "El Paso-Bocapotreronone (Zona 9) con | 27.00 | 55.00 |
| Creación: | | |
| Se crea la plaza de Auxiliar 2° en la Oficina mixta de Puente Nacional (Departamento de Santander), con | $ 55.00 | |
| y se nombra para desempeñarla a Luis F. León Moreno. | | |

**Artículo 2°** El presente Decreto regirá desde la fecha de su aprobación.

Comuníquese y publíquese.

Dado en Bogotá a 27 de enero 1942.

## EDUARDO SANTOS

El Ministro de Hacienda y Crédito Público,

## Carlos LLERAS RESTREPO

El Ministro de Correos y Telégrafos,
