# DECRETO 640 DE 1962

(marzo 19)

### Por medio del cual se modifíca el artículo 1° del Decreto número 2687 de noviembre 23 de 1960

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales, y en desarrollo de los Decretos números 0550, 1648 , 1678 y 1679 de 1660

### DECRETA:

**Articulo primero.** Sustituyese la denominación y grado los siguientes cargos, incorporados al Ministerio del Trabajo por Decreto número 2687 de 1960

| | | |
| --- | --- | --- |
| Despacho del Ministro | | |
| Grado | | |
| 1 | Secretario Privado II | 8 |
| 1 | Secretario Auxiliar II | 6 |
| Por: | | |
| 1 | Secretario Privado IV | 10 |
| 1 | Secretario Auxiliar IV | 8 |

**Artículo segundo.** Suprímese el siguiente cargo en la planta de personal del mismo Ministerio:

*Grupo de Relaciones Internacionales del*

*Trabajo.*

| | | |
| --- | --- | --- |
| *Grado* | | |
| 1 | Traductor I | 7 |

**Artículo tercero.** El presente Decreto rige a partir de la fecha de su expedición, pero surte efectos fiscales a partir del 16 de febrero del corriente año.

Comuníquese y cúmplase.

Dado en Bogotá, D. E., a 14 de marzo de 1962.
