# DECRETO 1063 DE 1903

(noviembre 18)

### Por el cual se restablece una Oficina Telegráfica y Postal, se reorganiza otra y se fijan las asignaciones que deben tener

**Subtipo:** DECRETO ORDINARIO

***El Vicepresidente de la República, encargado del Poder Ejecutivo***,

En uso de sus facultades legales,

### DECRETA:

| | | |
| --- | --- | --- |
| **Art. 1.°** Restablécese la Oficina Telegráfica y Postal de Simaña (Departamento del Magdalena), con el siguiente personal y asignaciones mensuales: | | |
| Un Telegrafista Administrador de Correos | $ | 1,600 |
| Un Cartero | | 200 |
| Para gastos de útiles de escritorio de correos | | 30 |

| | | |
| --- | --- | --- |
| **Art. 2.°** Reorganízase la Oficina Postal y Telegráfica de Ráquira (Departamento de Boyacá), la que prestará el servicio de tralación con el siguiente personal y asignaciones mensuales: | | |
| Un Telegrafista Administrador de Correos | $ | 2,000 |
| Un Ayudante | | 1,800 |
| Un Cartero | | 300 |
| Para gastos de útiles de escritorio de correos | | 30 |

Comuníquese y publíquese

Dado en Bogotá, á 18 de Noviembre de 1903.

### JOSE MANUEL MARROQUÍN

El Ministro de Gobierno,

### Esteban Jaramillo
