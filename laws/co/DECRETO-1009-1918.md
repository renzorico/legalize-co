# DECRETO 1009 DE 1918

(julio 02)

### Por el cual se reorganizan unas oficinas y se hacen unos nombramientos en el ramo Telegráfico

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### decreta:

**Articulo 1.º** Reorganízase las Oficinas Postales y Telegráficas de Churidó y Micuro, cada una con el personal y asignaciones mensuales siguientes:

| |
| --- |
| **Churidó** |

| | | |
| --- | --- | --- |
| Un Telegrafista Administrador de Correos, con | $ | 50 |
| Para gastos de alumbrado | | 3 |
| **Micuro** | | |
| Un Telegrafista Administrador de Correos, con | 40 | |
| Para gastos de alumbrado | 1 | |

| | | |
| --- | --- | --- |
| **Artículo 2.º** Nómbrase Telegrafistas Administradores de Correos de Churidó y Micuro, respectivamente, a los señores Jesús García y Rafael Restrepo. | | |

**Artículo 3.º** Mientras se encarga Restrepo, nómbrase internamente Telegrafista Administrador de correos Micuro al señor Julio Laverde.

Comuníquese y publíquese.

Dado en Bogotá a 2 de julio de 1918

### JOSE VICENTE CONCHA**-El Ministro de Gobierno, **Miguel ABADIA MENDEZ.
