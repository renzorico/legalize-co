# DECRETO 983 DE 1953

(abril 16)

### Por el cual se fija la dirección técnico- administrativa del Instituto de enfermedades Tropicales " Roberto Franco ", de Villavicencio, y se señala el personal, asignaciones y gastos de sostenimiento del mismo

**Subtipo:** DECRETO ORDINARIO

### El Designado, encargado de la Presidencia de la República de Colombia,

en uso de sus facultades legales y en especial de las que le confiere el Decreto extraordinario número 3123 de 1952,

### DECRETA:

**Artículo primero.** A partir del 16 de abril del presente año la dirección técnico-administrativa al Instituto de Enfermedades Tropicales "Roberto Franco", de Villavicencio, estará a cargo del Instituto de Estudios Especiales "Carlos Finlay".

**Artículo segundo.** A partir de la fecha indicada en el artículo anterior, el personal, asignaciones y gastos de sostenimiento del Instituto de Enfermedades Tropicales "Roberto Franco" de Villavicencio, será el siguiente:

| | | |
| --- | --- | --- |
| | **mensuales** | |
| 1 Director | $ | 1.000.00 |
| 1 Jefe de Investigaciones | | 700.00 |
| 1 Entomólogo | | 700.00 |
| 1 Ayudante de Parasitología | | 350.00 |
| 1 Ayudante de Investigaciones | | 350.00 |
| 1 Ayudante de Campo | | 200.00 |
| 1 Mecánico Carpintero | | 300.00 |
| 1 Oficial de Vacunación | | 200.00 |
| 1 Celador | | 160.00 |
| Para equipos, materiales, viáticos, jornales y demás gastos de sostenimiento | | 2.000.00 |

**Artículo tercero.** El Ministerio de Higiene por medio de Resoluciones posteriores, fijará las normas que sean necesarias para el cumplimiento del presente Decreto.

**Artículo cuarto.** : Los gastos qua demande el cumplimiento, del presente Decreto, se imputarán al Capítulo 64, artículo 734 del Presupuesto Nacional de Gastos de la vigencia en curso.

**Artículo quinto.** Los empleados cuyas denominaciones no hayan cambiado no requieren de nuevo nombramiento ni posesión.

**Artículo sexto.** Queda derogado el Decreto número 2595 de 1950 y demás disposiciones contrarias al presente Decreto.

Afecta la vigencia de: []

Comuníquese y publíquese.

Dado en Bogotá a 16 de abril de 1953.
