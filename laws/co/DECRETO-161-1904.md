# DECRETO 161 DE 1904

(febrero 19)

### Por el cual se restablece una Oficina

**Subtipo:** DECRETO ORDINARIO

## El Vicepresidente de la República, encargado del Poder Ejecutivo,

En uso de sus facultades legales,

**DECRETA**:

**Artículo único.** Restablécese en Chámeza la Administración de Correos nacionales, desde el 1 de Marzo próximo entrante, con el siguiente personal y asignaciones mensuales oro:

| | |
| --- | --- |
| Un Administrador de Correos | $ 8 |
| Para gastos de útiles de escritorio | $ 0,50 |

| | |
| --- | --- |
| Comuníquese y publíquese. Dado en Bogotá, á 19 de Febrero de 1904. **JOSE MANUEL MARROQUÍN** El Ministro de Gobierno, **ESTEBAN JARAMILLO** | |
