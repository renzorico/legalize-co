# DECRETO 2637 DE 1952

(octubre 23)

### Por el cual se distribuye la partida señalada para sostenimiento de la Escuela Normal Regular de señorita, de Ocaña (Norte de Santander)

**Subtipo:** DECRETO ORDINARIO

### El Designado, encargado de la Presidencia de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

**Artículo único.** Para sueldos del presonal de la Escula Normal Regular de Señoritas, de Ocaña (Norte de Santander), distribúyese la siguiente partida apropiada en el artículo 1° del Decreto legislativo número 2271 de 24 de septiembre de 1952, así:

*Capítulo 24, artículo 9457-A.*

| | | | |
| --- | --- | --- | --- |
| | | *Mensual* | *Total* |
| Sueldos en 4 meses, setiembre a diciembre: | $ | 500.00 | 2.000.00 |
| Directora | | 300.00 | 1.200.00 |
| Secretaria Habilitada. | | 300.00 | 1.200.00 |
| Directora de la Escuela Anexa | | 220.00 | 880.00 |
| Subdirectora de la Escuela América | | 1.040.00 | 4.160.00 |
| Cuatro profesoras internas directoras de grupo, a $ 260 cada una | | 140.00 | 560.00 |
| Total | | $ | 10.000.00 |

**Parágrafo.** La partida de $ 17.065,25 que aporta el Departamento Norte de Santander para la Escuela Normal Regular de Señoaritas, de Ocaña, según contrato de nacionalización, será distribuida por Resolución del Ministerio de Educación con destino a los demás gastos de sostenimiento del plantel.

Comuníquese y publíquese.

Dado en Bogotá a 23 de octubre de 1952.

### ROBERTO URDANETA ARBELAEZ

El Ministro de Hacienda y Crédito Público,
