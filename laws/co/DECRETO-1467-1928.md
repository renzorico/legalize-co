# DECRETO 1467 DE 1928

(agosto 11)

### Por el cual se distribuye una partida

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus atribuciones legales,

### DECRETA :

**Artículo único.** Distribúyese la partida de $ 16,829, saldo del artículo 711, capítulo 53 del Presupuesto de la actual vigencia, Educación Física, así :

| | |
| --- | --- |
| 1.° Para ayudar a la construcción del Stadium en Cali | $ 4,000 |
| 2.° Para la Olimpíada Nacional (premios y demás gastos ordinarios de la Olimpíada) | 4,000 |
| 3.° Para gastos de la Sección | |
| 4.ª (sueldos, útiles y traslación de la Sección a Cali a la Olimpiada Nacional) | 8,829 |
| Suma | $ 16,829 |

| | |
| --- | --- |
| Comuníquese y publíquese. Dado en Bogotá a 11 de agosto de 1928. **MIGUEL ABADIA MENDEZ** El Ministro de Educación Nacional, **J. Vicente HUERTAS** | |
