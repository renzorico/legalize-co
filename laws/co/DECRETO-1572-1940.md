# DECRETO 1572 DE 1940

(agosto 12)

### Por el cual se aprueba, con modificaciones, el Acuerdo número 11 de 1940, julio 29, dictado por el Consejo Administrativo de la Intendencia Nacional del Chocó

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia.

en uso de sus facultades,

### DECRETA:

**Artículo único**. Apruébase el siguiente Acuerdo:

### ACUERDO NUMERO 11 DE 1940

(julio 29)

### por el cual se modifican los Acuerdos números 4 y 7 de 1940, sobre presupuesto intendencial de rentas y Acuerdo de Apropiaciones para el año fiscal del primero de enero al 31 de diciembre de 1940.

*El Consejo Administrativo de la Intendencia del Chocó,*

en uso de sus atribuciones legales,

### ACUERDA:

Artículo 1° De las apropiaciones que a continuación se expresan, por concepto de sobrantes de sueldos no devengados o de partidas que no tendrán aplicación en el presente año, témanse, para trasladar, las siguientes cantidades:

| | |
| --- | --- |
| *Del capítulo III-Departamento de Hacienda:* | |
| Artículo 38. Sueldo del Jefe de la Fábrica de Licores de Istmina, de febrero a junio, cinco meses, a Si 90 mensuales | $ 450.00 |
| *Del capítulo VI-Departamento de Obras Públicas:* | |
| Artículo 136. Para construcción de acueductos | 1.500.00 |
| Total que se toma | $ 1.950.00 |

| | |
| --- | --- |
| **Artículo 2°** Tómase el total de mil novecientos cincuenta pesos ($1.950) moneda Corriente para aumentar apropiaciones que quedaron deficientes conforme al siguiente pormenor: | |
| *Al capítulo I-Departamento de Gobierno:* | |
| Artículo 16. Para terminar las instalaciones de las estaciones radiotelegráficas y pago de sueldos en enero | 1.500.00 |
| Suma el Departamento de Gobierno | $ 1.500.00 |
| *Al capítulo III-Departamento de Hacienda:* | |
| Artículo 39. Para compra de licores, alcoholes, mieles, anís, conducción de éstos, adquisición de materiales, gastos de explotación, de destilación, pago de jornales y otros gastos | $ 450.00 |
| Suma el Departamento de Hacienda | $ 450.00 |
| *Recapitulación.* | |
| Departamento de Gobierno | $ 1.500.00 |
| Departamento de Hacienda | 450.00 |
| Suma igual | $1.950.00 |

**Artículo 3°** El presente Acuerdo requiere para su validez la aprobación del Organo Ejecutivo Nacional.

Dado en Quibdó a 29 de julio de 1940.

El Secretario de Gobierno, encargado,

*Gerardo García Gómez*

El Oficial del Despacho Intendencial,

*Abigail Arriaga V.*

El suscrito Secretario del Consejo Administrativo de la Intendencia del Chocó, certifica que el presente Acuerdo sufrió dos debates en sesiones distintas, v en ambas fue aprobado Quibdó, julio 30 de 1940.

*Abigail Arriaga* V., Oficial del Despacho Intendencial,
