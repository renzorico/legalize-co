# DECRETO 178 DE 1958

(mayo 28)

### Por el cual se hacen unas traslaciones en el Presupuesto de Gastos vigente. (Ministerio de Agricultura y de Obras Públicas)

**Subtipo:** DECRETO LEGISLATIVO

### La Junta Militar de Gobierno de Colombia,

obrando de conformidad con las disposiciones del artículo 121 de la Constitución Nacional,

### DECRETA:

**Artículo primero**. Hácense las siguientes traslaciones en el Presupuesto de Gastos vigente:

MINISTERIO DE AGRICULTURA

CAPITULO 204

### División de Extensión

| | | |
| --- | --- | --- |
| **Construcción y Dotación.** | | |
| 235. Del artículo 2121. Para atender a los gastos que demande la construcción y dotación de edificios en las Granjas de Tibaitatá y Palmira, para el Servicio Nacional de Extensión Agrícola | $ | 250.000.00 |
| Suma el contracrédito | $ | 250.000.00 |

MINISTERIO DE OBRAS PUBLICAS

CAPITULO 584

CARRETERAS NACIONALES

| | | |
| --- | --- | --- |
| **Construcción, Reconstrucción y Pavimentación.** | | |
| 232. Al artículo 5910-A. Para la carretera Puerto Porfía-Cabuyaro, en los Llanos Orientales. | $ | 250.000.00 |
| Suma el crédito | $ | 250.000.00 |

**Artículo segundo**. El presente Decreto rige desde la fecha de su expedición y suspende todas las disposiciones que le sean contrarias.

Comuníquese, publíquese y cúmplase.

Dado en Bogotá a 28 de mayo de 1958.
