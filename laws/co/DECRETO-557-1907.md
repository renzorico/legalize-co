# DECRETO 557 DE 1907

(mayo 11)

### Por el cual se reorganiza el personal del crucero Cartagena, se le señala sueldo y se eleva la partida para gastos de material

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia

### DECRETA:

**Artículo 1°.** En lo sucesivo el personal del crucero Cartagena será el siguiente, con las asignaciones mensuales que se expresan:

| --- | --- |
| Comandante | $ 200 -- |
| Capitán náutico | 150 -- |

| Un segundo Oficial Contador | 60 -- |

| Un Piloto | 60 -- |
| Un primer Contramaestre | 35 -- |
| Un segundo Contramaestre | 30 -- |
| Dos Timoneles, cada uno | 25 -- |
| Un Carpintero | 25 -- |
| Quince Marinaros, cada uno | 14 -- |

| Un Sargento 1.° | 27 -- |
| Diez Cabos jefes de piezas, cada uno | 16 -- |
| Veinticinco individuos de tropa, cada uno | 14 -- |

| | |
| --- | --- |
| **Artículo 2°.** Nómbrase Comandante del crucero *Cartagena* en propiedad al señor Coronel Daniel Angulo. | |

**Artículo 3°.** Autorízase al Coronel Angulo para hacer los nombramientos del personal de su dependencia perteneciente al crucero Cartagena, de conformidad con el presente Decreto y dando cuenta al Ministerio de Guerra para su aprobación.

**Artículo 4°.** En caso de un viaje largo, el Comandante del crucero Cartagena podrá conceder primas á los fogoneros, ó aumentar el número de ellos si fuere necesario.

**Artículo 5°.** Elévase á cuatrocientos pesos ($ 400) la partida mensual para gastos de material del crucero Cartagena, los cuales serán pagados por anticipación.

Dado en Bogotá, á 11 de Mayo de 1907.

### R. REYES

E1 Subsecretario de Guerra encargado del Despacho,

**CLIMACO**LOSADA
