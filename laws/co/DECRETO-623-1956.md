# DECRETO 623 DE 1956

(marzo 16)

### Por el cual se distribuye una partida

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

**Artículo único** La partida trasladada por Decreto 3171 de diciembre 7 de 1955 al Capítulo 103, artículo 18/7 del Presupuesto de 1955 para sueldos y. alimentación del personal de la Escuela Normal de Rurales para Señoritas, de Sibundoy (Nariño), se distribuye así:

| | | |
| --- | --- | --- |
| *Sueldos en 3 meses, octubre a diciembre:* | | |
| *Mensual* | *Total* | |
| De 2 profesoras de Normal a .$ 350.00 cada una | $ 700.00 | 2.100.00 |
| *Alimentación en 3 meses, octubre a diciembre:* | | |
| De 2 profesoras internas de Normal a $ 45,00 cada una | 90.00 | 270.00 |
| Total | $ 790.00 | 270.00 |

**Parágrafo** La partida distribuida en el artículo anterior será girada al Secretario Habilitado del plantel

Comuníquese y publíquese

Dado en Bogotá a 16 de marzo de 1956.

Gral. Jefe Supremo **GUSTAVO ROJAS PINILLA**

Presidente de Colombia

El Ministro de Hacienda y Crédito Público,

## Carlos Villaveces
