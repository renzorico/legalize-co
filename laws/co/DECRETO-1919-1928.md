# DECRETO 1919 DE 1928

(octubre 09)

### Por el cual se reorganiza la agencia postal de Buenaventura

**Subtipo:** DECRETO ORDINARIO

### POR EL CUAL SE REORGANIZA LA AGENCIA POSTAL DE BUENAVENTURA

### El Presidente de la República de Colombia

en uso de sus facultades legales,

### DECRETA:

**Artículo 1.°** Reorganízase la Agencia Postal de Buenaventura con el personal y asignaciones mensuales siguientes:

### AGENCIA POSTAL

| | |
| --- | --- |
| Un Agente, con | $ 300 |
| Un Oficial, con | 120 |
| Un Auxiliar, con | 110 |

### SECCION DE RECIBO Y DISTRIBUCION DE CORREOS

| | |
| --- | --- |
| Un Jefe, con | 150 |
| Un Oficial, con | 120 |
| Cinco Ayudantes, a $ 70 cada uno | 350 |
| Dos Carteros, a $ 60 cada uno | 120 |

### SECCION DE ENCOMIENDAS DEL EXTERIOR

| | |
| --- | --- |
| Un Jefe, con | 150 |
| Dos Auxiliares, a $ 110 cada uno | 220 |
| Cinco Guardas, a $ 60 cada uno | 300 |

### SECCION DE RECOMENDADOS, VALORES DECLARADOS Y ENCOMIENDAS DEL INTERIOR

| | |
| --- | --- |
| Un Jefe, con | 150 |
| Un Oficial, con | 120 |
| Un Auxiliar, con | 110 |
| Valor mensual | $ 2,320 |

**Artículo 2.º** el presente Decreto regirá desde el día en que se apropie en el Presupuesto la partida necesaria para atender al gasto que su ejecución demanda.

Comuníquese y publíquese.

Dado en Bogotá a 9 de octubre de 1928.

### MIGUEL ABADIA MENDEZ

El Ministro de Correos y Telégrafos,

### José Jesús GARCIA
