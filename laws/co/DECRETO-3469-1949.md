# DECRETO 3469 DE 1949

(noviembre 02)

### por el cual se reforma un Decreto.

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales, y

CONSIDERANDO

que habiendo quedado debidamente organizado el Almacén del Instituto de Enfermedades Tropicales "Roberto Franconone, de Villavicencio, se justifica que la Pagaduría quede adscrita al cargo de Almacenista,

### DECRETA:

**Artículo primero.** A partir del primero de octubre del corriente año, suprímese el cargo de Administrador Pagador del Instituto de Enfermedades Tropicales "Roberto Franconone.

**Artículo segundo.** Adscribense al Almacenista Jefe de DDT las funciones de Pagador del mismo Instituto, a partir del primero de octubre del presente año.

**Artículo tercero.** A partir del primero de octubre del presente año, los cargos, asignaciones y gastos de sostenimiento del Instituto "Roberlo Franconone, quedarán así:

| | | |
| --- | --- | --- |
| | *Mensuales* | |
| Director | $ | 800.00 |
| Jefe de Investigaciones | | 700.00 |
| Pagador Almacenista y Jefe de DDT | | 550.00 |
| Oficial de Bio-estadística | | 350.00 |
| Ayudante de Parasitología | | 350.00 |
| Inspector Vacunador | | 200.00 |
| Oficial de Fumigación | | 200.00 |
| Ayudante de investigaciones | | 200.00 |
| Carpintero | | 200.00 |
| Para administración, equipos, sostenimientos, jornales, viáticos y pago de materiales adquiridos | | 4.486.97 |

**Artículo cuarto.** El presente Decreto no altera el gasto mensual imputable al Capítulo 54, artículo 707 del Presupuesto Nacional vigente.

**Artículo quinto.** Para efectos del cambio de asignaciones, el Pagador Almacenista y Jefe de DDT, Ayudante de Investigaciones y Carpintero del Instituto "Roberto Franconone, requieren nueva posesión.

Comuníquese y cúmplase.

Dado en Bogotá a 2 de noviembre de 1949.
