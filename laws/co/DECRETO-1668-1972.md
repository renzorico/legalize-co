# DECRETO 1668 DE 1972

(septiembre 12)

### Por los cuales se modifica la posición 38. 19. R. del Arancel de Aduanas

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en ejercicio de las facultades conferidas por la Ley 6ª de 1971, y

considerando:

Que mientras los antibióticos tienen asignada una tarifa del 1%, la materia prima básica para su obtención -micelios de fermentación- está gravada con el 50%;

Que con el objeto de corregir esta situación anacrónica que obstaculiza el desarrollo normal de la industria nacional de antibióticos, se consideró necesario modificar la posición 38.19.R. del Arancel de Aduanas;

Que el Consejo Nacional de Política Aduanera se pronunció favorablemente sobre la modificación que aquí se señala,

### decreta:

**Artículo 1°** El texto y gravámenes de la posición 38.19.R. será el siguiente:

| | | |
| --- | --- | --- |
| **Posición** | **Denominación** | **Gravamen** |
| | | % |
| 38.19. | Productos químicos y preparados de las industrias químicas o de los industrias conexas (incluidos los que consisten en mezclas de productos naturales), no expresados ni comprendidos en otras posiciones; productos residuales de las industrias químicas o de las industrias conexas, no expresados ni comprendidos en otras posiciones: | |
| | R. Productos de la fabricación de antibióticos: | |
| | I. Micelios inactivos (tortas alcalinas), con un contenido mínimo del 5% de antibiótico activo | 1 |
| | II. Otros | 20 |

**Artículo 2º** Este Decreto rige a partir de la fecha de su expedición.

Comuníquese y cúmplase.

Dado en Bogotá, D. E., a 12 de septiembre de 1972.

### misael pastrana borrero

El Ministro de Hacienda y Crédito Público, **Rodrigo Llorente Martínez.**
