# DECRETO 1388 DE 1919

(julio 07)

### Por el cual se reorganizan unas oficinas en el ramo Telegráfico

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### decreta:

Reorganízase las Oficinas Telegráficas de Dabeiba y Frontino, cada una con el personal y asignaciones mensuales siguientes:

| | | |
| --- | --- | --- |
| **Dabeiba** | | |
| Un Telegrafista Administrador de Correos, con | $ | 40 |
| Un Ayudante, con | | 35 |
| Un Cartero, con | | 5 |
| Para gastos de alumbrado | | 3 |
| **Frontino** | | |
| Un Telegrafista Administrador de Correos, con | | 35 |
| Un Cartero, con | | 5 |
| Para gastos de alumbrado | | 1 |

El presente Decreto empezará a regir diez días después de su fecha.

Comuníquese y publíquese.

Dado en Bogotá a 7 de julio de 1919.

### MARCO FIDEL SUAREZ**-El Ministro de Gobierno, **Marcelino ARANGO.
