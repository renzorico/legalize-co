# DECRETO 1649 DE 1938

(septiembre 13)

### Por el cual se reorganiza el Departamento de Minas

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso ele sus atribuciones constitucionales y especialmente de las que le confiere la Ley 96 del presente año,

### DECRETA:

**Artículo 1.°** El Departamento de Minas del Ministerio de la Economía Nacional tendrá el personal y las asignaciones que en seguida se expresan:

| | | |
| --- | --- | --- |
| Director | $ | 300.00 |
| Abogado | | 250.00 |
| Tres Ingenieros, cada uno a | | 209.00 |
| Secretario | | 120.00 |
| Archivero | | 100.00 |
| Ayudante | | 70.00 |
| Cuatro Mecanógrafos, cada uno a | | 70.00 |
| Conserje, Ayudante del Archivero | | 70.00 |

**Artículo 2.°** Hácense los siguientes nombramientos,

Director del Departamento, doctor Juan Donoso Gómez.

Abogado, doctor Arturo Valleio S.

Ingeniero, doctor Virgilio Madrid.

Ingeniero, doctor Nicolás Rosso R. ingeniero, doctor Luis González.

Secretario señor Ramón Castaño.

Mecanógrafo, señorita María E. Triana.

Mecanógrafo, señorita Emma Salgado.

Mecanógrafo,.señorita Oliva Navas.

Mecanógrafo, señora Ana Rosa de Castilla.

Archivero, señora Ana v. de García.

Conserje, Ayudante del Archivero, señor Luis F. Hencker.

**Artículo tercero.** Nómbrase miembro de la Comisión Revisora del Código de Minas al doctor Luis Andrés Gómez, con la asignación mensual de $ 250.00

Comuníquese y publíquese.

Dado en Bogotá, a 13 de septiembre de 1933.
