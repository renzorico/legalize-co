# DECRETO 1239 DE 1919

(junio 20)

### Por el cual se reorganizan unas oficinas y se hacen unos nombramientos en el ramo Telegráfico

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### decreta:

**Artículo 1°** Reorganízanse las Oficinas Telegráficas de El Cedro y Valdivia, cada una con el personal y asignaciones mensuales siguientes:

| | | |
| --- | --- | --- |
| **El Cedro.** | | |
| Un Telegrafista Administrador de Correos, con | $ | 30 |
| Para gastos de alumbrado | | 1 |
| **Valdivia.** | | |
| Un Telegrafista Administrador de Correos | | 40 |
| Un Ayudante | | 32 |
| Un Cartero | | 3 |
| Para gastos de alumbrado | | 3 |

**Artículo 2°** Nómbrase Telegrafista Administrador de Correos de El Cedro, al señor Francisco Hernández, y a la Señorita Oliva Galindo y el señor Hernando Cuartas, Jefe y Ayudante de la Oficina Postal y Telegráfica de Valdivia respectivamente.

El presente Decreto empezará a regir cinco días después de su fecha.

Comuníquese y yublíquese.

Dado en Bogotá a 20 de junio de 1919.

### MARCO FIDEL SUAREZ**-El Ministro de Gobierno, **Marcelino ARANGO.
