# DECRETO 698 DE 1983

(marzo 08)

### Por el cual se modifica la planta de personal del Departamento Administrativo Nacional de Estadística

**Subtipo:** DECRETO ORDINARIO

El Presidente de la República de Colombia, en ejercicio de sus facultades constitucionales y legales, especialmente las que le confiere el ordinal 21 del artículo 120 de la Constitución Nacional,

DECRETA:

**Artículo 1°**Suprímanse en la planta de personal del Departamento Administrativo Nacional de Estadística, los siguientes cargos:

| | | | |
| --- | --- | --- | --- |
| **N° de Cargos** | **Dependencia** **Denominación del cargo** | **Código** | **Grado** |
| | JEFATURA DEL DEPARTAMENTO | | |
| 1 Uno | Chofer mecánico | 6010 | 06 |
| | SECRETARIA GENERAL | | |
| 1 Uno | Profesional Especializado | 3010 | 10 |

**Articulo 2°**Créase en la planta de personal del Departamento Administrativo Nacional de Estadística el siguiente cargo:

| | | | |
| --- | --- | --- | --- |
| **N° de Cargos** | **Dependencia** **Denominación del cargo** | **Código** | **Grado** |
| | JEFATURA DEL DEPARTAMENTO | | |
| 1 Uno | Asesor | 1020 | 02 |

Artículo 3º El presente Decreto rige a partir de la fecha do su expedición.

Comuníquese y cúmplase.

Dado en Bogotá, D. E., a 8 marzo de 1983.
