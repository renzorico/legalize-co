# DECRETO 1244 DE 1943

(junio 23)

### Se suprimen unos cargos en las reparticiones de la Fuerza Aérea

**Subtipo:** DECRETO ORDINARIO

### en uso de sus facultades legales y de la contenida en el inciso 2° del artículo 25 de La Ley 45 de 1942.

### DECRETA:

**Artículo 1°** Suprímase los siguientes cargos en las reparticiones de la Fuerza Aérea que a continuación se expresan:

a) *Dirección General*

Departamento de Aeronáutica Civil:

1 Mecanotaquígrafo.

b) *Base Aérea de Madrid*

Sección de Operaciones:

1 Jefe Servicio de estadística.

**Artículo 2°** En lugar de los puestos suprimidos por el artículo 1° del presente decreto, créanse los siguientes, así:

| | | |
| --- | --- | --- |
| *Dirección General*. | | |
| Sección de OperacionesnoneServicio estadística: | | |
| 1 Adjunto 3° | $ | 100.00 |
| *Base Aérea de Madrid.* | | |
| Sección de Operaciones: | | |
| 1 Mecanógrafo 1° | | 70.00 |

**Artículo 3°** El Presente Decreto surtirá sus efectos con fecha 1° de julio del año en curso.

Comuníquese y cúmplase.

Dado en Bogotá a 23 de junio de 1943.

## ALFONSO LOPEZ

El Ministro de Guerra,

## Ramón SANTO DOMINGO
