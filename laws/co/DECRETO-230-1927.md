# DECRETO 230 DE 1927

(febrero 12)

### Por el cual se modifica el marcado con el número 2021 de 1926, reorgánico de la sección primera del ferrocarril central del Norte

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### decreta:

**Artículo 1°** Suprímense los siguientes puestos en la sección primera del ferrocarril central del Norte:

Inspector General de construcción, Secretario del Ingeniero Jefe de la Empresa, Secretarios de los Ingenieros de Sección y primer Dibujante de la Oficina General de Ingeniería.

**Artículo 2°** Créanse los siguientes cargos en la misma empresa:

| | | | |
| --- | --- | --- | --- |
| Un Ingeniero de replanteo con. | $ | 350 | mensuales |
| Un Ayudante del Ingeniero Jefe de construcción, con | | 250 | |
| Un encargado de la Oficina del trabajo, con | | 100 | |
| Un Ingeniero Ayudante del Ingeniero Jefe de la Oficina General de Ingeniería, con | | 280 | |
| Cuatro Ingenieros Ayudantes de los Ingenieros de sección de construcción, cada uno con | | 250 | |
| Un Médico de línea, residente en Bucaramanga, con | | 150 | |

**Artículo 3°** Fíjanse como asignaciones mensuales de los empleados que se expresan a continuación, las siguientes:

| | | |
| --- | --- | --- |
| Pagador de línea | $ | 250 |
| Jefe de tráfico | | 180 |
| Inspector de herramientas | | 120 |
| Segundo Dibujante de la Oficina General de Ingeniería | | 200 |
| Terceros Dibujantes de la misma, cada uno con | | 170 |

Comuníquese y publíquese.

Dado en Bogotá a 12 de febrero de 1927.

### MIGUEL ABADIA MENDEZ

El Ministro de Obras Públicas, **Mariano OSPINA PEREZ.**
