# DECRETO 20 DE 1915

(enero 05)

### por el cual se reorganizan unas Oficinas en el ramo Telegráfico.

**Subtipo:** DECRETO ORDINARIO

### *El Presidente de la República de Colombia*****,

En uso de sus facultades legales,

**decreta**:

Reorganízanse las Oficinas Postales y Telegráficas de Medialuna y Piñón, cada una con el personal y asignaciones mensuales siguientes:

### Medialuna.

| | | |
| --- | --- | --- |
| Un Telegrafista Administrador de Correos, con | $ | 25 |
| Un Cartero | | 2 |
| Un Telegrafista Administrador de Correos, con | $ | 50 |
| Un Ayudante. | | 45 |
| Un Cartero | | 4 |

Comuníquese.

Dado en Bogotá a 5 de enero de 1015.

### JOSE VICENTE CONCHA

El Ministro de Gobierno,

### MIGUEL ABADIA MENDEZ.
