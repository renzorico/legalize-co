# DECRETO 1079 DE 1942

(abril 25)

### Por el cual se hacen unas modificaciones en el personal del ramo de Telégrafos en el Ministerio de Correos y Telégrafos

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

**Artículo 1°** Hácense las siguientes modificaciones en el personal del ramo de Telégrafos en el Ministerio de Correos y Telégrafos :

| | | |
| --- | --- | --- |
| *Supresiones:* | | |
| Se suprime una plaza de Auxiliar 3° de la Oficina Telegráfica de Bolívar (Departamento de Antioquia) desempeñada por Germán Vélez Escobar, con asignación mensual de | $ | 50.00 |
| Se suprime la plaza de Auxiliar 3° de la Oficina Mixta de Circasia, con asignación mensual de | | 45.00 |
| | $ | 95.00 |
| *Creación:* | | |
| Se crea la plaza de Mecánico Revisor de la Oficina Telegráfica de Ipiales, con asignación mensual de | $ | 90.00 |
| y se nombra para desempeñarla a José D. Muñoz. | | |
| *Aumento:* | | |
| Se aumenta la asignación mensual del Telegrafista Administrador de Cispatá, en | | 5.00 |
| | $ | 95.00 |

**Artículo 2°** El presente Decreto regirá 10 días después de su aprobación.

Comuníquese y publíquese.

Dado en Bogotá a 25 de abril de 1942.

## EDUARDO SANTOS

El Ministro de Hacienda y crédito Público,

## Carlos LLERAS RESTREPO

El Ministro de Correos y Telégrafos,

## Luis BUENAHORA
