# DECRETO 2255 DE 1948

(julio 01)

### por el cual se crean varios cargos en el ramo de Carreteras Nacionales.

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus atribuciones legales, y en especial de las que le confieren los artículos 7° y 15 de la Ley 88 de 1931,

### DECRETA :

**Artículo primero.** Para la carretera Popayán-Guapi se crean los cargos que a continuación se indican, con las siguientes remuneraciones mensuales:

| | |
| --- | --- |
| a) Un Inspector de Trabajos, con | $ 270.00 |
| b) Un Secretario Ayudante de Contabilidad, con | 150.00 |
| c) Un Farmacéutico Enfermero, con | 120.00 |
| d) Un Jefe de Maquinaria y Bulldozero, con | 300.00 |

| | |
| --- | --- |
| **Artículo segundo.** En la carretera Villeta-Guaduas-Honda se crean los siguientes cargos: | |
| a) Un Inspector de Trabajos, con | $ 270.00 |
| b) Un Subinspector Apuntatiempo, con | 200.00 |
| c) Un Secretario Ayudante de Contabilidad, con | 150.00 |
| d) Un Farmacéutico Enfermero, con | 180.00 |
| e) Un Jefe de Talleres, con | 300.00 |
