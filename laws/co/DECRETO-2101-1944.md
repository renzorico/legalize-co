# DECRETO 2101 DE 1944

(septiembre 08)

### Por la cual se hacen unos traslados en el presupuesto vigente (Ministerio de Guerra y Obras Públicas)

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales, y

CONSIDERANDO

que el Consejo de Ministros en su sesión del día 19 de agosto del año en curso, autorizó al Ministerio de Hacienda y Crédito Público para efectuar unos traslados en la Ley de Apropiaciones vigente, según consta en el acta respectiva,

| |
| --- |
| **DECRETA:** |

| |
| --- |
| **Artículo 1º.** Hácense los siguientes traslados en el Presupuesto vigente: |
| PRESUPUESTO EXTRAORDINARIO DEL MINISTERIO DE GUERRA |

| | | |
| --- | --- | --- |
| Capítulo 104 bis. Del artículo 1926-M. Para construcciones en 3a Base Aérea "Germán Olano" | $ | 70.000.00 |
| PRESUPUESTO EXTRAORDINARIO | | |
| AL MINISTERIO DE OBRAS PÚBLICAS | | |
| Capítulo 116. Al artículo 1999. Para construcciones en la Base Aérea "German Olano" | $ | 70.000.00 |

**Artículo 2º.** El presente Decreto rige desde sil fecha..

Comuníquese y publíquese.

Dado en Bogotá a 8 de septiembre de 1944.

### ALFONSO LOPEZ

El Ministro de Hacienda y. Crédito Público,

### Gonzalo RESTREPO

El Ministro dé Guerra;

### Domingo****ESPINEL

El Ministro de Obras Públicas
