# DECRETO 649 DE 1937

(marzo 16)

### Por el cual se hacen unos traslados en el presupuesto de la Comisaría de Arauca

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades,

### decreta:

**Artículo 1°** Hácense los siguientes traslados dentro del presupuesto actual de la Comisaría Especial de Arauca:

| | | |
| --- | --- | --- |
| DEPARTAMENTO DE HACIENDA | | |
| **Capítulo II.** | | |
| Del artículo 10. Para materias primas, conducción de licores, material y demás gastos de la fábrica | $ | 1,000.00 |
| Del artículo 12. Para compra de un aparato de destilación | | 1,500.00 |
| DEPARTAMENTO DE OBRAS PÚBLICAS | | |
| **Capítulo V.** | | |
| Del artículo 27. Para dar principio a la construcción del edificio para Colegio de varones de Arauca | | 500.00 |
| Del artículo 28. Para obras públicas de Arauquita, de acuerdo con el plan que forme el Comisario | | 200.00 |
| Del artículo 29. Para las obras públicas en Tame, según el plan que elabore el Comisario | | 200.00 |
| Suman las partidas tomadas | $ | 3,400.00 |

Artículo 2° La anterior suma, de tres mil cuatrocientos pesos ($ 3,400), se traslada así:

| | | |
| --- | --- | --- |
| Al artículo 23, capítulo 5°, Departamento de Obras Públicas. Para construcción del puente de Las Cañadas | | 3,400.00 |
| Suma el traslado | $ | 3,400.00 |

Comuníquese y publíquese.

Dado en Bogotá a 16 de marzo de 1937.

### ALFONSO LOPEZ

El Ministro de Agricultura y Comercio,

### Manuel José VARGAS
