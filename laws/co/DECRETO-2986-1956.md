# DECRETO 2986 DE 1956

(diciembre 06)

### Por el cual se distribuye una partida (Ramo de Carreteras Nacionales)

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

**Artículo único.** De la partida apropiada en el Capítulo 530, artículo 5376 del Presupuesto vigente, "para carreteras y puentes de la Intendencia del Caquetá, según distribución que hará el Gobierno Nacional", hácese las siguiente distribución:

| | | |
| --- | --- | --- |
| CAPITULO 530 | | |
| **Artículo 5376** |
| Ordinal 3). Para el puente El Encanto, sobre-el río Hacha | $ | 100.000.00 |
| Ordinal 4). Para el puente Morelia sobre el río Bodoquero | | 50.000.00 |
| Suma | $ | 150.000.00 |

Comuníquese y publíquese.

Dado en Bogotá a 6 de diciembre de 1956.

General Jefe Supremo **GUSTAVO ROJAS PINILLA.**

El Presidente de Colombia.

El Ministro de Hacienda y Crédito Público,

### Luis Morales Gómez.

El Ministro de Obras Públicas,

Contraalmirante **Rubén Piedrahita Arango.**
