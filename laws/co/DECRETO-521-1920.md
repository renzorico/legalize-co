# DECRETO 521 DE 1920

(marzo 04)

### Por el cual se crean unas oficinas en el ramo Telegráfico

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### decreta:

Créanse las Oficinas Postales y Telegráficas de Palmarito (Departamento de Bolívar, Distrito de Majagual), Badenia y San Bernardo (Departamento de Cundinamarca), cada una con el personal y asignaciones mensuales siguientes:

### Palmarito.

| | | |
| --- | --- | --- |
| Un Telegrafista Administrador de Correos, con | $ | 40 |
| Un Cartero | | 5 |
| Para gastos de alumbrado | | 1 |

### Badenia.

| | |
| --- | --- |
| Un Telegrafista Administrador de Correos, con | 30 |
| Un Cartero | 5 |
| Para gastos de alumbrado | 1 |

### San Bernardo.

| | |
| --- | --- |
| Un Telegrafista Administrador de Correos, con | 30 |
| Un Cartero | 5 |
| Para gastos de alumbrado | 1 |

Comuníquese y publíquese.

Dado en Bogotá a 4 de marzo de 1920.

Por delegación del Excelentísimo señor Presidente, el Ministro de Gobierno, **Luis** CUERVO MARQUEZ.
