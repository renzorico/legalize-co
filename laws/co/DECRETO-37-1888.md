# DECRETO 37 DE 1888

(enero 16)

### Por el cual se reorganiza el Resguardo de Ciénaga Grande y sus alrededores

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República

**DECRETA** :

**Art. 1.°** El Resguardo de Ciénaga Grande y sus alrededores constará de los siguientes empleados :

| | |
| --- | --- |
| Un Jefe con sueldo anual de | $ 900 |

| | |
| --- | --- |
| Un Cabo con id. id. de | 480 |

| | |
| --- | --- |
| Seis Guardas con id. id. cada uno de | 300 |

| | |
| --- | --- |
| Cuatro Remeros con id. id. id. id. de | 240 |

En estos sueldos están comprendidos los gastos de movilización que tengan que hacer por razón del servicio.

**Art. 2.°** Nómbrase Jefe del expresado Resguardo al Sr. José R. Juvinao ;

Cabo, al Sr. Joaquin Zúñiga Rosa :

Guardas, á los Sres. José María Salcedo, Manuel J. Alvarez, Mercedes Fuertes, Mercedes Fuertes S., José Dolores Ibarra y Cosme Altamarinda ; y

Romeros, á los Sres. Ismael Aponte. Manuel Bolaño, Manuel Villafaña y Andrés Díaz.

Dado en Bogotá, á 16 de Enero de 1888.

**ELISEO PAYÁN**.

El Ministro de Hacienda,

### Vicente Restrepo.
