# DECRETO 1137 DE 1992

(julio 07)

### por el cual se suprimen unos cargos y se crea un cargo en la planta de personal Interno del Ministerio de Relaciones Exteriores.

**Subtipo:** DECRETO CONSTITUCIONAL

El Presidente de la República de Colombia, en uso de las facultades que le confiere el numeral 14 del artículo 189 de la Constitución Política,

### DECRETA:

**Artículo 1º** Suprímanse los siguientes cargos de la Planta de Personal del Servicio Interno del Ministerio de Relaciones Exteriores establecida mediante Decreto 2924 de 31 diciembre de 1991:

| | | | |
| --- | --- | --- | --- |
| **Número de cargos** | **Dependencia y denominación del empleo** | **Código** | **Grado** |
| **Despacho del Viceministro Relaciones Exteriores** | | | |
| 1 | Asesor | 1020 | 04 |
| **Secretaría General** | | | |
| 1 | Profesional Especializado | 3010 | 14 |

Afecta la vigencia de: []

**Artículo 2º** Créase el siguiente cargo en la Planta de Personal del Servicio Interno del Ministerio de Relaciones Exteriores de conformidad con lo establecido en el artículo 2º numeral 2º de la Ley 11 de 1991.

| | | | |
| --- | --- | --- | --- |
| **Número de cargos** | **Dependencia y denominación del empleo** | **Código** | **Grado** |
| **Despacho del Viceministro de Asuntos Políticos Internacionales** | | | |
| 1 | Viceministro | 0020 | |

**Artículo 3º** El presente Decreto rige a partir de la fecha de su publicación y modifica en lo pertinente el Decreto 2924 de 1991.

Afecta la vigencia de: []

Publíquese, comuníquese y cúmplase.

Dado en Santafé de Bogotá, D.C., a 7 de julio de 1992.
