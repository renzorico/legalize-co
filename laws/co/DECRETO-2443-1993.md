# DECRETO 2443 DE 1993

(diciembre 07)

### por el cual se modifica el Decreto 47 del 7 de enero de 1993 y se dictan otras disposiciones.

**Subtipo:** DECRETO ORDINARIO

El Presidente de la República de Colombia, en desarrollo de las normas generales señaladas en la Ley 4ª de 1992,

### DECRETA:

**Artículo 1°** Adiciónase la escala de remuneración para los empleados del Nivel Ejecutivo de la Registraduría Nacional del Estado Civil, así:

| | |
| --- | --- |
| **Escala del Nivel Ejecutivo** | |
| **Grado** | **Asignación básica** |
| 11 | 602.046 |
| 12 | 648.255 |
| 13 | 694.485 |

**Artículo 2°** Adiciónase la nomenclatura de empleos de la Registraduría Nacional del Estado Civil, establecida en el Decreto 89 del 4 de enero de 1990, así:

| | | |
| --- | --- | --- |
| **Denominación** | **Código** | **Grado** |
| **Nivel Ejecutivo** | | |
| Director | 2030 | 13 |
| | | 12 |
| | | 11 |

**Artículo 3°** El presente Decreto rige a partir de la fecha de su publicación, modifica en lo pertinente los Decretos 89 de 1990 y 47 de 1993 y deroga las disposiciones que le sean contrarias.

Afecta la vigencia de: []

Publíquese y cúmplase.

Dado en Santafé de Bogotá, D. C., a 7 de diciembre de 1993.

### CESAR GAVIRIA TRUJILLO
