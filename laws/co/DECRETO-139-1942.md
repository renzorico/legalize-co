# DECRETO 139 DE 1942

(enero 23)

### Por el cual se distribuyen las partidas para sueldos del personal y alimentación de alumnos y empleados internos del Instituto Isidro Parra, del Líbano

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en uso de sus facultades legales,

DECRETA:

**Artículo único.** Para sueldos del personal y alimentación de alumnos y empleados internos del Instituto "Isidro Parranone, (Colegio Oficial del Líbano), se distribuyen las siguientes partidas apropiadas en el Presupuesto Nacional vigente, así:

| | | |
| --- | --- | --- |
| A) Capítulo 62. artículo 865. | | |
| Para sueldos en enero, así: | | |
| | Mensual. | Total. |
| Nueve Profesores, Directores de Grupo (internos), a $ 130 mensuales cada uno | $ 1.170.00 | 1.170.00 |
| Profesor de Educación Física | 100.00 | 100.00 |
| Portero | 15.00 | 15.00 |
| Suma | $ 1.285.00 | 1.285.00 |
| Para sueldos en once (11) meses, de febrero a diciembre, así: | | |
| Nueve Profesores, Directores de Grupo (internos), a $ 130 mensuales cada uno | $ 1.170.00 | 12.870.00 |
| Profesor de Educación Física, a | 100.00 | 1.100.00 |
| Portero, a | 20.00 | 220.00 |

| Para alimentación en dos (2) meses, enero y diciembre, así: | | |
| Alimentación de cuatro (4) empleados internos, a $ 14 mensuales cada uno | $ 56.00 | 112.00 |

| Alimentación de diez y nueve (19) empleados internos, a $ 14 cada uno | 266.00 | 2.660.00 |
| Servicios: | | |
| Para servicios de energía eléctrica, agua, aseo, material de enseñanza, útiles de escritorio, deportes, compra y reparación de mobiliario y demás gastos del establecimiento | | 5.228.00 |
| Total | | $ 8.000.00 |
| **Parágrafo.** La partida de quince mil pesos ($ 15.000.00) que corresponde aportar al Departamento del Tolima, según el contrato de nacionalización del Colegio Oficial del Líbano (Instituto Isidro Parra), será distribuida por resolución del Ministerio de Educación Nacional, destinándola a cubrir los demás gastos del mencionado establecimiento. | | |

Comuníquese y publíquese.
