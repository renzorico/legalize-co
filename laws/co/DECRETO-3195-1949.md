# DECRETO 3195 DE 1949

(octubre 12)

### Por el cual se modifican algunas disposiciones del Decreto 3036 de 1949

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de las facultades que le confiere la Ley 93 de 1948,

DECRETA:

**Artículo 1°** A partir de la vigencia del presente Decreto, suprímese el siguiente personal del fijado para la Policía Nacional, por Decreto 3036 de 1949:

| | | |
| --- | --- | --- |
| 1 Subcomisario | $ | 350.00 |
| 1 Comisario a prueba | | 280.00 |
| 1 Subinspector | | 220.00 |
| 1 Adjunto de Nóminas de la Caja General | | 220.00 |

| | | |
| --- | --- | --- |
| **Artículo 2°** A partir de la misma fecha, créanse los siguientes cargos en la Sección VI del Departamento de Servicios Administrativos, con las asignaciones mensuales que se expresan: 1 Cajero Auxiliar | $ | 400 00 |
| 1 Contador 2° | | 350.00 |
| 1 Oficial de Nóminas | | 280.00 |

| | | |
| --- | --- | --- |
| **Artículo 3°** Acláranse los artículos 5° y 6° del Decreto 3036 de 1949, en el sentido de que el Jefe de Personal y el Director General de los Servicios, son del grado de Comandante y no de Subcomandante, como allí se dice. | | |

**Artículo 4°** Este Decreto rige desde su fecha.

Comuníquese y cúmplase.

Dado en Bogotá a 12 de octubre de 1949.

## MARIANO OSPINA PEREZ

El Ministro de Gobierno,

## Luis Ignacio ANDRADE

El Ministro de Hacienda y Crédito Público,
