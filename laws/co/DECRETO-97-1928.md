# DECRETO 97 DE 1928

(enero 19)

### Por el cual se suprimen unos puestos en el ramo Telegráfico y se dispone la partida a que asciende ese ahorro para dar cumplimiento al Decreto número 1992, de fecha 7 de diciembre próximo pasado, sobre reorganización de las Oficinas de Cali y de La Dorada

**Subtipo:** DECRETO ORDINARIO

en uso de sus facultades legales,

**decreta**:

**Artículo 1°** Suprímanse los siguientes puestos de las Oficinas Telegráficas que a continuación se expresan:

### Bogotá.

| | |
| --- | --- |
| Un Operador de la Central, con $170 mensuales, en el año | $2040 |

### Chiquinquirá.

| | |
| --- | --- |
| Un Ayudante, con $ 89 mensuales, en el año | 3068 |

### Girardot.

| | |
| --- | --- |

| Pasan | $4464 |
| Vienen | $4464 |

### Magangué.

| | |
| --- | --- |
| dos Ayudantes, con $113 mensuales, en el año | 2712 |

### Sonsón,

| | |
| --- | --- |
| Un Telegrafista, con $ 60 mensuales, en el año | 720 |
| Suma total de ahorros | 7896 |

**Artículo 2°** Para der cumplimiento al Decreto 1992, de fecha 7 de diciembre próximo, pasado, por el cual se reorganizan las Oficinas Telegráficas de Cali y de La Dorada, destínase la suma de siete mil cuatrocientos veinticinco pesos con veinte centavos ($ 7,425 20), a que asciende el mayor valor del gasto conforme al citado Decreto, y el que figura en el Presupuesto de la actual vigencia, capítulo 56, artículo 732. Esta suma se tomará del ahorro que se hace por el presente Decreto,

Comuníquese y publíquese,

Dado en Bogotá a 19 de enero de 1928.
