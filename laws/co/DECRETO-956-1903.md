# DECRETO 956 DE 1903

(octubre 22)

### Por el cual se restablecen dos oficinas Telegráficas

**Subtipo:** DECRETO ORDINARIO

## El Vicepresidente de la República, encargado del Poder Ejecutivo,

En uso de sus facultades legales,

### DECRETA:

**Artículo único.** Restablécense las Oficinas Telegráficas de San Cayetano (Departamento de Cundinamarca) y Santa Ana (Departamento de Boyacá), anexas á las Administraciones de Correos, cada un con el siguiente personal y asignaciones mensuales:

| | | |
| --- | --- | --- |
| *San Cayetano* | | |
| Un Telegrafista Administrador de Correos | $ | 2,000 |
| Un Cartero | | 300 |
| Para útiles de escritorio de Correos | | 20 |
| *Santa Ana* | | |
| Un Telegrafista Administrador de Correos | $ | 1,400 |
| Un Cartero | | 200 |
| Para útiles de escritorio de Correos | | 20 |

Comuníquese y publíquese.

Dado en Bogotá, á 22 de Octubre de 1903.

### JOSE MANUEL MARROQUÍN

El Ministro de Gobierno,

### ESTEBAN JARAMILLO
