# DECRETO 424 DE 1957

(marzo 01)

### Por el cual se hace la distribución de una partida

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

**Artículo primero**. De la partida asignada en el Capítulo 206, artículo 2385 del Presupuestó Nacional vigente, "para becas de las Facultades de Agronomía y Veterinaria, en el país, en el año", hácese la siguiente distribución:

| | | |
| --- | --- | --- |
| CAPITULO 206 **Artículo 3386-Ordinal 1°** | | |
| Aporte del Gobierno Nacional para el pago de 52 becas, a razón de § 1.200.00 anuales cada una, durante año lectivo de 1957, a la Facultad de Agronomía de Medellín | $ | 62.400.00 |
| CAPITULO 206 **Artículo 2385-Ordinal 1** | | |
| Aporte del Gobierno Nacional para el pa de 49 becas, a razón de $ 1.200.00 anuas cada una, durante el año lectivo de 1957, la Facultad de Agronomía de Palmira | | 58.800.00 |
| CAPITULO 206 **Artículo 3385-Ordinal 3** | | |
| Aporte del Gobierno Nacional para el pa de 38 becas, a razón de $ 1.200.00 anual cada una, durante el año lectivo de 157, a la Facultad de Agronomía de Mázales | | 45.600.00 |
| CAPITULO 206 **Artículo 2385-Ordinal 4** | | |
| Aporte del Gobierno Nacional para el pa de 29 becas, a razón de $ 1.200.00 cada la, anuales, durante el año lectivo de 1957, la Facultad de Veterinaria de Manizales | | 34.800.00 |
| CAPITULO 206 **Artículo 2385-Ordinal 5** | | |
| Aporte del Gobierno Nacional para el pa de 40 becas, a razón de $ 1.200.00 anual cada una, durante el año lectivo de 1957, la Facultad de Veterinaria de Bogotá | | 48.000.00 |

**Artículo segundo**. El pago de los aportes ordenados el artículo 1° de este Decreto, su manejo e inversión la rendición de cuentas correspondientes a la Contraria General de la República, se efectuarán con arreo a las normas administrativas y fiscales vigentes para esta clase de erogaciones.

Comuníquese, publíquese y cúmplase.

Dado en Bogotá a 1° de marzo de 1957.

General Jefe Supremo **GUSTAVO ROJAS PINILLA,**

Presidente de Colombia.

El Ministro de Hacienda y Crédito Público,
