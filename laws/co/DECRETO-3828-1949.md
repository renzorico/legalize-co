# DECRETO 3828 DE 1949

(diciembre 01)

### por el cual se hacen unas supresiones y unas creaciones en el personal del ramo de Telégrafos del Ministerio de Correos y Telégrafos.

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

*en uso de sus atribuciones legales, y en especial de la conferida por el parágrafo 3° del artículo 1° de la Ley 66 de 1926,*

### DECRETA:

**Artículo 1°** Hácense las siguientes supresiones en el personal del ramo de Telégrafos del Ministerio de Correos y Telégrafos:

| | | |
| --- | --- | --- |
| DEPARTAMENTO DE CALDAS | | |
| *Arauca.* | | |
| La plaza de Telegrafista Administrador, con asignación mensual de | $ | 100.00 |
| Cargo que viene desempeñando el señor Humberto Munar. | | |
| INTENDENCIA NACIONAL DEL META | | |
| *Puerto López.* | | |
| La plaza de Auxiliar Segundo de la Oficina Mixta, con asignación mensual de | | 149.00 |
| Cargo que viene desempeñando la señora María Livia Buitrago de Núñez. | | |
| Suman | $ | 249.00 |

**Parágrafo.** Las partidas anteriores corresponden al Capítulo 98; artículo 2003, del Presupuesto vigente.

**Artículo segundo.** Hácense las siguientes creaciones en el personal del ramo de Telégrafos del Ministerio de Correos y Telégrafos:

| | | |
| --- | --- | --- |
| DEPARTAMENTO DE CALDAS | | |
| *Risaralda.* | | |

| INTENDENCIA NACIONAL DEL META | | |
| *Acacias-Corregimiento.* | | |
| Una plaza de Auxiliar Segundo de la Oficina Mixta, con asignación mensual de | | 149.00 |
| Suman | $ | 249.00 |

**Parágrafo.** El gasto que demande el cumplimiento de lo dispuesto en el artículo anterior se imputará al Capítulo 98, articulo 2003, del Presupuesto vigente.

**Artículo tercero.** El presente Decreto rige a partir del día primero (1°) de diciembre del año en curso.

Comuníquese y cúmplase.

Dado en Bogotá a 1° de diciembre de 1949.
