# DECRETO 1705 DE 1915

(octubre 13)

### por el cual se reorganizan dos Oficinas y se crean unos empleos en el ramo Telegráfico.

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### decreta:

| | | |
| --- | --- | --- |
| | Bolívar | |
| Un Telegrafista Administrador de Correos, con | $ | 40 |
| Un Ayudante con | | 30 |
| Un cartero con | | 5 |
| | Carmen | |
| Un Telegrafista Administrador de | | 30 |
| Un Cartero | | 4 |

**Artículo 1º** Reorganízanse las Oficinas Postales y Telegráficas de Bolívar (Antioquia) y Carmen (Atrato), cada una con el personal y asignaciones mensuales siguientes:

**Artículo 2º** Créase la plaza de Ayudante en la Oficina Telegráfica de Barbacoas, con la asignación mensual de $ 40, y una más en la de Cúcuta, con la misma asignación de las de su clase en aquella Oficina.

Comuníquese y publíquese,

Dado en Bogotá a 13 de octubre de 1915.

### JOSE VICENTE CONCHA**. ­El Ministro de Gobierno, **MIGUEL ABADÍA MÉNDEZ.
