# DECRETO 2003 DE 1915

(diciembre 04)

### por el cual se crean unas Oficinas en el ramo Telegráfico.

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República,

en uso de sus facultades,

**decreta**:

Créanse las Oficinas Telegráficas de Yacopí (Departamento de Cundinamarca), Almaguer, y Puracé (Departamento del Cauca), cada una con el personal y asignaciones siguientes:

| | | |
| --- | --- | --- |
| Yacopi: | | |
| Un Telegrafista Administrador de Correos, con | $ | 22 |
| Un cartero | | 3 |
| Pagos para alumbrado | | 1 |
| Almaguer: | | |
| Un Telegrafista Administrador de Correos, con | $ | 30 |
| Un cartero | | 3 |
| Pagos para alumbrado | | 1 |
| Puracé: | | |
| Un Telegrafista Administrador de Correos, con | $ | 25 |
| Un cartero | | 3 |
| Pagos para alumbrado | | 1 |

Comuníquese y publíquese.

Dado en Bogotá a 4 de diciembre de 1915.

**JOSE VICENTE CONCHA**-El Ministro de Gobierno, **Miguel** ABADIA MENDEZ.
