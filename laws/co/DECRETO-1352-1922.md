# DECRETO 1352 DE 1922

(septiembre 20)

### Por el cual se reorganizan unas oficinas y se hacen unos nombramientos en el ramo Telegráfico

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### decreta:

**Artículo 1°** Reorganízanse las Oficinas Telegráficas de Carnicerías y La Plata, cada una con el personal y asignaciones mensuales siguientes:

### Carnicerías.

| | | |
| --- | --- | --- |
| Un Telegrafista Administrador de Correos, con. $ | | 40 |
| Un Ayudante, con | | 30 |
| Un Cartero, con | | 5 |
| Para gastos de alumbrado | | 2 50 |
| **La Plata.** Un Telegrafista, con | | 40 |
| Un Cartero, con | | 7 |
| Para gastos de alumbrado | | 1 50 | |

| | | |
| --- | --- | --- |
| **Artículo 2°** Nómbrase Telegrafista de La Plata al señor Pastor Cuenca, y Jefe y Ayudante de la Oficina Postal y Telegráfica de Carnicerías a los señores Rafael Losada y Romualdo Ortiz, respectivamente. | | |

**Artículo 3°** El presente Decreto empezará a regir desde el primero de octubre próximo. Comuníquese y publíquese.

Dado en Bogotá a 20 de septiembre de 1922.

### PEDRO NEL OSPINA**-El Ministro de Gobierno, **Miguel JIMENEZ LOPEZ.
