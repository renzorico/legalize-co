# DECRETO 1492 DE 1959

(mayo 25)

### Por el cual se modifica la distribución de la partida global apropiada para sueldos del personal de la Escuela Normal Superior de Varones de Corozal

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en ejercicio de sus atribuciones legales, y especialmente de las que le confiere el artículo 8° de la Ley 35 de 1958,

### DECRETA:

**Artículo primero.** Modifícase la distribución de la partida apropiada para sueldos del personal de la Escuela Normal Superior de Varones de Corozal, hecha por Decreto número 0482 de 18 de febrero de 1959, en la siguiente forma:

| | | | |
| --- | --- | --- | --- |
| *Capítulo 388, artículo 3973, ordinal 23.* | | | |
| Hácese la siguiente supresión: | | *Mensual* | *Total* |
| Once meses, febrero a diciembre | | | |
| Un Profesor interno Director de Grupo, de cuarta categoría o sin escalafón | $ | 450.00 | 4.950.00 |
| La suma anteriormente mencionada se distribuye así: | | | |
| Para incrementar la partida para profesorado externo, en el año | $ | | 4.950.00 |

**Artículo segundo.** Queda en estos términos modificado el artículo primero del Decreto número 0482 de 18 de febrero de 1959.

Comuníquese y publíquese.

Dado en Bogotá, D. E., a 25 de mayo de 1959.

### ALBERTO LLERAS

El Ministro de Hacienda y Crédito Público,

## Hernando Agudelo Villa

El Ministro de Educación Nacional,
