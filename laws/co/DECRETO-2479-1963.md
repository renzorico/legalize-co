# DECRETO 2479 DE 1963

(octubre 15)

### Por el cual se cancela una partida, se hace una distribución parcial de la destinada al sostenimiento de becas nacionales, y se efectúa una aclaración

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### DECRETA:

**Artículo primero.** De la partida distribuida según Decreto Presupuestal número 583 de 21 de marzo de 1963, para el sostenimiento de becas nacionales en el año de 1962, hácense las siguientes cancelaciones:

### En tres (3) meses, de octubre a diciembre

| | | | |
| --- | --- | --- | --- |
| | | **Mensual** | **Total** |
| 18. Becas en la Normal de Señoritas de Almaguer (Cauca), | | | |
| a $ 80 mensuales cada una | | 1.440.00 | 4.320.00 |
| 20 Becas en la Escuela Vocacional Agrícola de Ipapure, | | | |
| a $ 80.00 mensuales cada una | | 1.600.00 | 4.800.00 |
| | | | 9.120.00 |

**Artículo segundo.** De la partida apropiada en el Capítulo número 1208, artículo 12081 del presupuesto nacional de 1962, para el sostenimiento de becas nacionales, destinase la suma de $ 9.120.00:

En tres (8) meses, de octubre a diciembre:

| | | | |
| --- | --- | --- | --- |
| | | **Mensual** | **Total** |
| 18. Becas en la Normal de Señoritas de Popayán (Cauca), | | | |
| a $ 80 mensuales cada una | | 1.440.00 | 4.320.00 |
| 20 Becas en la Escuela Vocacional Agrícola de Buga (Valle), | | | |
| a $ 80 mensuales cada una | | 1.600.00 | 4.800.00 |
| | | | 9.120.00 |

**Artículo tercero.** Aclárase el artículo primero del decreto 1058 de abril 27 de 1962, en el sentido de que las becas que figuran presupuestadas para el Colegio "Laurentina Moreno", del Socorro (Santander), deberán pagarse al Colegio "Avelina Moreno", del Socorro (Santander), que es su verdadero nombre.

Comuníquese y cúmplase.

Dado en Bogotá, D. E., a octubre 15 de 1963.

### GUILLERMO LEON VALENCIA

El Ministro de Hacienda y Crédito Público, **Carlos Sanz de Santamaría.** El Ministro de Educación Nacional, **Pedro Gómez Valderrama.**
