# DECRETO 1101 DE 1920

(mayo 26)

### Por el cual se reorganizan unas oficinas, se suprime una plaza y se crean otras en el ramo Telegráfico

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia

en uso de sus facultades legales,

### decreta:

**Artículo 1.none** Reorganízanse las Oficinas Telegráficas de Calará y Filandia, cada una con el personal y asignaciones mensuales siguientes:

| | | |
| --- | --- | --- |
| **Calarcá** | | |
| Un Telegrafista Jefe, con | $ | 60 |
| Un Ayudante, con | | 55 |
| Un Cartero | | 6 |
| **Filandia** | | |
| Un Telegrafista Administrador de Correos | 35 | |
| Un Cartero | 6 | |

**Artículo 2.º** Suprímese una plaza de Ayudante en la Oficina Telegráfica de Vílleta, y créase una más en la de Honda, con la misma asignación mensual de las de su clase en esta Oficina.

El presente Decreto empezará a regir diez días después de su fecha.

Comuníquese y publíquese.

Dado en Bogotá a 26 de mayo de 1920.

**MARCO FIDEL SUAREZ**-El Ministro de Gobierno, **Luis CUERVO MARQUEZ**.
