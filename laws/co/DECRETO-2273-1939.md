# DECRETO 2273 DE 1939

(noviembre 29)

### Por el cual se distribuye una partida

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en uso de sus facultades legales,

### DECRETA :

**Artículo único.** El saldo de seis mil novecientos sesenta pesos con cuarenta y ocho centavos ($ 6.960.48), de la apropiación correspondiente al artículo 365, capítulo 58, del Presupuesto vigente, destinado para el sostenimiento de la Escuela Normal de varones de Quibdó, se distribuye así:

| | | |
| --- | --- | --- |
| *Sueldos en diciembre:* | | |
| Del Rector | $ | 250.00 |
| Del Vicerrector | | 160.00 |
| De cuatro Profesores internos, a $ 130 cada uno | | 520.00 |
| Del profesorado externo, a $ 2 la hora de clase | | 350.00 |
| Del Profesor de Física y Química | | 150.00 |
| Del Secretario Habilitado | | 150.00 |
| De la Directora de la Anexa | | 100.00 |
| De la Subdireetora de la Anexa | | 80.00 |
| Del Médico Profesor de la Normal y la Anexa | | 80.00 |
| Del Portero | | 30.00 |
| Del personal de servicio | | 230.00 |
| Para pagar al señor Edmundo Quiñones sus servicios como Secretario Habilitado en los últimos nueve días de septiembre | | 45.00 |
| *Alimentación:* | | |
| Pensiones alimenticias del Rector, Secretario y Portero, a $ 22 cada uno | | 66.00 |
| Para pagar lo que se adeuda por concepto de víveres suministrados en el corriente año | | 3.811.43 |
| *Material y varios:* | | |
| Para arrendamiento, reparaciones, drogas y demás gastos de material, suministrados a la Escuela, en el año | | 938.05 |
| Total | $ | 6.960.48 |

**Parágrafo.** En estos términos quedan modificados los Decretos números 166 y 1810 de 1939.

Afecta la vigencia de: []
