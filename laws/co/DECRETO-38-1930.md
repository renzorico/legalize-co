# DECRETO 38 DE 1930

(enero 11)

### POR EL CUAL SE CREA UNA OFICINA POSTAL Y TELEGRAFICA EN EL DEPARTAMENTO DE BOYACA, Y SE HACE UN NOMBRAMIENTO

**Subtipo:** DECRETO ORDINARIO

### POR EL CUAL SE CREA UNA OFICINA POSTAL Y TELEGRAFICA EN EL DEPARTAMENTO DE BOYÁCA, Y SE HACE UN NOMBRAMIENTO

**El** Presidente de la República de Colombia,

en uso de sus facultades legales,

DECRETA:

**Artículo 1°** Créase la Oficina Postal y Telegráfica intermedia de Sátivasur, en el Departamento de Boyacá, con el personal y asignaciones siguientes:

| | | |
| --- | --- | --- |
| Un Telegrafista Administrador de Correos, con sueldo mensual de | $ | 50 |
| Para gastos de alumbrado, mensualmente | | |

| | | |
| --- | --- | --- |
| **Artículo 2°** Nómbrase al señor Vicente Dávila Telegrafista Administrador de Correos de Sátivasur. | | |

**Artículo 3°** El gasto que demande el cumplimiento del presente. Decreto se imputará al artículo 684, capitulo 35, del Presupuesto vigente.

Comuníquese y publíquese.

Dado en Bogotá a 11 de enero de 1930.

### MIGUEL ABADIA MENDEZ

El Ministro de Correos y Telégrafos,

### José Jesús GARCIA

Digitó: 52.741.900
