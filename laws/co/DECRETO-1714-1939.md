# DECRETO 1714 DE 1939

(agosto 29)

### Por el cual se modifica el artículo 1º del Decreto número 131 y el único del Decreto número 661 de 1939

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en uso de sus atribuciones legales,

### DECRETA:

**Artículo 1°** De los fondos apropiados en el artículo 331, capítulo 57, del Presupuesto vigente, distribuyese la cantidad de veinte mil seiscientos siete pesos con treinta y siete centavos ($ 20.607.37) moneda corriente, para gastos del Curso de Información Pedagógica Nacional, en los meses de septiembre a diciembre, así:

*Sueldos*-En 4 meses, de septiembre a diciembre, así:

| | | |
| --- | --- | --- |
| | Mensual. | Total. |
| Del Director del Curso | $ 250,00 | 1.000.00 |
| Del Subdirector, encargado del Curso Nacional de Extensión Cultural por correspondencia | 180.00 | 720.00 |
| De dos Mecanógrafas, a $ 60 mensuales cada una | 120.00 | 480.00 |
| Del Encargado del mimeógrafo | 55.00 | 220.00 |
| De tres Profesores permanentes para Ciencias Naturales, Historia y Geografía, y Ciencias Pedagógicas, a S 150 cada uno | 450.00 | 1.800.00 |
| De cuatro Profesores de Prácticas Pedagógicas, a $ 150 cada uno | 600 00 | 2.400.00 |
| De cuatro Profesoras de Prácticas Pedagógicas, a $ 120 mensuales cada una. | 480.00 | 1.920.00 |
| De tres Sirvientes, a $ 25 mensuales cada uno | 75.00 | 300.00 |
| Para pagar 113 horas de clase mensual, a $ 2.50 cada hora | 282.50 | 1.130.00 |
| *Becas*-En tres meses, de septiembre a noviembre, así: | | |
| Para pagar nueve becas, a 8 40 mensuales cada una | 360.00 | 1.080.00 |
| | Mensual. | Total. |
| Para pagar sesenta y dos becas, a $ 30 mensuales cada una | 1.860.00 | 5.580.00 |

| | | |
| --- | --- | --- |

| Total | | $ 20.607.37 |

**Artículo 2°** En estos términos quedan modificados los artículos 1° del Decreto número 131 y único del Decreto número 661 de 1939.
