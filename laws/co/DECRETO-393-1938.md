# DECRETO 393 DE 1938

(marzo 02)

### Por el cual se adiciona el Decreto número 172 de 1938

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

**Artículo único.** Adiciónase la distribución que se hizo por medio del artículo del Decreto número 172 del corriente año, de los fondos votados en el artículo 305, Capítulo 46, de la Ley de Apropiaciones vigente, para sueldos del Instituto Pedagógico Nacional, con las siguientes partidas:

| | | | | |
| --- | --- | --- | --- | --- |
| | **Mensual** | **En 11 meses** | | |
| Sueldo de la Ecónoma | $ | 60.00 | $ | 660.00 |
| Sueldo de la profesor a de oficios domésticos | | 30.00 | | 330.00 |
| Total | | | $ | 990.00 |

Afecta la vigencia de: []

Comuníquese y publíquese.

Dado en Bogotá a 2 de marzo de 1938.
