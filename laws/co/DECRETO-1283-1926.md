# DECRETO 1283 DE 1926

(julio 29)

### Por el cual se determinan los signos convencionales y las abreviaturas militares para el Ejército

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de facultades legales,

**decreta**:

**Artículo 1º** Los signos militares convencionales para el Ejército, serán los siguientes, los cuales se llenarán con color azul para las propias tropas, y con rojo para las tropas enemigas:

Caballería.

Patrulla.

Pelotón.

Escuadrón.

Pelotón de ametralladoras.

Escuadrón de ametralladoras.

Artillería.

Pieza de campaña.

Ingenieros.

Comunicaciones.

Compañía de telegrafistas.

Compañía de radiotelegrafía.

Aviación.

Avión.

Escuadrilla.

Grupo de aviación.

Tren.

Columna de camiones automóviles.

Límites de zona.

Cuerpo de Ejército.

División o Brigada.

Regimiento.

**Artículo 3º** Designación de Unidades.

Sólo las Unidades fundamentales se designan con una cifra romana; las demás formaciones, por una cifra árabe, así:

| | |
| --- | --- |
| División y Brigada | .1 Cifra árabe. |
| Regimiento | 1 " ". |
| Batallón. Grupo | 1 " ". |
| Compañía, escuadrón, batería, columna, escuadrilla (Unidades fundamentales) | 1 Cifra romana. |
