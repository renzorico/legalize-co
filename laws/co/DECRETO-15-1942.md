# DECRETO 15 DE 1942

(enero 08)

### Por el cual se señalan las zonas en que se divide la red de carreteras nacionales en servicio, se fijan las vías que deben formar cada zona y se distribuye la partida apropiada en el Presupuesto vigente para la conservación de dicha red

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en uso de sus facultades legales, y

### TENIENDO EN CUENTA:

que para atender al sostenimiento de las carreteras terminadas últimamente es necesario incorporarlas a las zonas respectivas, en forma que consulte la mejor distribución de los trabajos y el conveniente reparto de las partidas que figuran en el articulo 1267 del Presupuesto Nacional para la vigencia en curso,

### DECRETA:

**Artículo 2º** La partida de tres millones novecientos cincuenta mil pesos ($ 3.950.000.00), apropiada en el articulo 1267 del Presupuesto Nacional para la vigencia en curso, para gastos de conservación de la red nacional de carreteras, se distribuirá así:

| Para la zona de Bogotá | $ 358.000.00 |
| Para la zona de Cartagena | 115.000.00 |
| Para la zona de Ibagué | 200.000.00 |
| Para la zona de Manizales | 235.000.00 |
| Para la zona de Medellín | 400.000.00 |
| Para la zona de Garzón | 230.000.00 |
| Para la zona de Pamplona | 285.000.00 |
| Para la zona de Pasto | 195.000.00 |

| Carretera Barbosa-Puerto Olaya | 50.000.00 |
| Carretera La Tagua-Caucaya | 36.000.00 |
| Para la compra de maquinaria para las diferentes zonas | 725.000.00 |
| Para estudios de pavimentación y contrato con la Universidad | 40.000.00 |
| Para gastos generales de carreteras | 25.000.00 |
| Total | $ 3.950.000.00 |

**Articulo 3º** Los Ingenieros y demás empleados de las zonas mencionadas, nombrados por decretos anteriores, a cuyo cuidado están la dirección y administración de los trabajos, continuarán en los mismos cargos que ejercen con los sueldos y atribuciones correspondientes a tales puestos.

Comuniquese y publiquese.
