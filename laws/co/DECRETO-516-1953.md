# DECRETO 516 DE 1953

(febrero 17)

### Por el cual se hacen unas traslaciones en el presupuesto de Gastos vigente (Ministerio de Guerra)

**Subtipo:** DECRETO ORDINARIO

### El Designado, enfardado de la Presidencia de la República de Colombia,

### en uso de sus atribuciones legales, y

### CONSIDERANDO:

Que el Consejo de Ministros en su sesión del día 25 de febrero del año en curso, autorizó al Ministerio de Hacienda y Crédito Público para efectuar unas traslaciones detro de las apropiaciones del Ministerio de Guerra, según consta en el acta respectiva,

### DECRETA:

**Artículo primero.** Hácense las siguientes traslaciones en el Presupuesto de Gastos vigente:

| | | |
| --- | --- | --- |
| MINISTERIO DE GUERRA. | | |
| **Capítulo 45.** | | |
| Ministerio y Ejército. | | |
| Del artículo 453. Para iniciar la construcción de los cuarteles del Batallón "Bárbula", en Soatá, en el año | $ | 250.000.00 |
| Suman los contracréditos | | 250.000.00 |
| MINISTERIO DE GUERRA | | |
| **Capítulo 46.** | | |
| Ministerio y Ejército. | | |
| Al artículo 447. Para continuar la construcción del cuartel y obras del "Batallón de Ingenieros Caldas", en Bogotá, en el año | $ | 250.000.00 |
| Suman los contracréditos | | 250.000.00 |

**Articulo segundo.** El presente Decreto rige desde su fecha.

Comuniqúese y publiquese.

Dado en Bogotá a 27 de febrero de 1953.

ROBERTO URDANETA ARBELAEZ

El Ministro de Hacienda y Crédito Público,

*Antonio Alvarez Restrepo*
