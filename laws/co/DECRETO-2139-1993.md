# DECRETO 2139 DE 1993

(octubre 27)

### por el cual se modifica la nomenclatura de empleos de que tratan los Decretos-leyes 1042 de 1978, 90 de 1988 y demás normas modificatorias.

**Subtipo:** DECRETO ORDINARIO

El Presidente de la República de Colombia, en desarrollo de las normas generales señaladas en la Ley 4ª de 1992,

### DECRETA:

**Artículo 1°** Adiciónase la nomenclatura de empleos de que tratan las Decretos-leyes 1042 de 1978, 90 de 1988 y demás normas modificatorias, así:

| | | |
| --- | --- | --- |
| **Denominación.** | **Código** | **Grado** |
| NIVEL DIRECTIVO | | |
| Director de Superintendencia | 0105 | 17 |
| | | 15 |
| | | 13 |
| Intendente | 0138 | 17 |
| | | 15 |
| Secretaria General de Superintendencia o Entidad Descentralizada | 0037 | 19 |
| | | 17 |
| Superintendente Delegado | 0110 | 19 |
| | | 18 |
| | | 17 |
| | | 15 |
| NIVEL EJECUTIVO | | |
| Director o Gerente Regional | 2035 | 25 |
| | | 23 |
| Jefe de Grupo | 2085 | 10 |
| NIVEL PROFESIONAL | | |
| Almacenista, Almacenista Laboratorista o Comprador de Empresa Industrial o Comercial | 3015 | 16 |

**Artículo 2°** La nomenclatura y clasificación de empleos de que tratan los Decretos extraordinarios 1042 de 1978, 90 de 1988 y demás normas modificatorias, rige para los empleos de las Empresas Industriales y Comerciales del Estado y de las Sociedades de Economía Mixta sometidos al régimen de dichas empresas, que deban ser desempeñados por empleados públicos.

**Artículo 3°** El presente Decreto rige a partir de la fecha de su publicación, adiciona los Decretos-leyes 1042 de 1978, 90 de 1988 y demás normas que los modifiquen.

Afecta la vigencia de: []

Publíquese y cúmplase.

Dado en Santafé de Bogotá, D. C., a 27 de octubre de 1993.
