# DECRETO 1622 DE 1994

(julio 27)

### por el cual se modifica la nomenclatura de empleos de que tratan los Decretos-leyes 1042 de 1978, 90 de 1988 y demas normas modificatorias.

**Subtipo:** DECRETO ORDINARIO

El Presidente de la República de Colombia, en desarrollo de las normas generales señaladas en la Ley 4º de 1992

### DECRETA:

Artículo 1ºAdicionáse la nomenclatura de empleos de que tratan los Decretos-leyes 1042 de 1978, 90 de 1988 y demás normas modificatorias, así:

| | | |
| --- | --- | --- |
| **Denominación** | **Código** | **Grado** |
| NIVEL DIRECTIVO | | |
| Gerente, Presidente o Director General | 0015 | 10 |
| de Establecimiento Público o | | |
| de Empresa Industrial | | 09 |
| y Comercial o asimilada o de Corporación | | 07 |
| Autónoma o de Empresa Social del Estado | | |
| **NIVEL TECNICO** | | |
| Asistente Administrativo | 4140 | 11 |

Artículo 2º El presente Decreto rige a partir de la fecha de su publicación, adiciona el Decreto-ley 1042 de 1978, y demás normas que los modifiquen.

Afecta la vigencia de: []

Publíquese y cúmplase.

Dado en Santafé de Bogotá, D.C., a 27 de julio de 1994.
