# DECRETO 1262 DE 1932

(julio 27)

### Por el cual se reconoce un mes de vacaciones al personal de la Escuela Normal de Varones de Pasto

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en uso de sus atribuciones legales, y

CONSIDERANDO

Que el Decreto número 652, de fecha 12 de abril del presente año, reconoció a favor de los Directores y Profesores de la Escuela Normal del Departamento de Nariño, sus sueldos de vacaciones remuneradas, únicamente para el mes de agoste próximo venidero;

Que el artículo 64 del Reglamento para las Escuetas Normales autoriza para concederles hasta dos meses de vacaciones remuneradas,

### DECRETA:

**Artículo único.** Reconócese a favor de los Directores y Profesores de la Escuela Normal del Departamento de Nariño, sus sueldos para el mes de septiembre de 1932, de conformidad con la siguiente liquidación:

| | | |
| --- | --- | --- |
| Del Director, a | $ | 100 |
| Del Subdirector, a | | 80 |
| Del Director de la Escuela Anexa, a | | 80 |
| Del Subdirector de la Escuela Anexa, a | | 60 |
| De dos Celadores Profesores, a $ 45 cada uno | | 90 |
| Del Portero, a | | 15 |
| De quince Profesores, a $ 20 cada uno | | 300 |
| Pensiones alimenticias de siete empleados internos, a $ 18 cada uno | | 126 |
| Suma | $ | 851 |
| **Parágrafo.** La partida para dar cumplimiento al presente Decreto se imputará al artículo 409, capítulo 49, del Presupuesto de la vigencia en curso. | | |

Dése cuenta a quienes corresponda, y publíquese.

Dado en Bogotá a 27 de julio de 1932.
