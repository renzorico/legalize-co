# DECRETO 2866 DE 1978

(diciembre 28)

### Por el cual se suprimen 24 cargos y se crean 22 cargos en la planta de personal del Departamento Administrativo Nacional de Estadística

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en ejercicio de sus facultades constitucionales y legales,

### decreta:

**Artículo 1°** Suprímanse en la planta de personal del Departamento Administrativo Nacional de Estadística, los siguientes cargos así:

| | | | |
| --- | --- | --- | --- |
| Dependencia Denominación del cargo | N° de cargo | Código | Grado |
| DIRECCION GENERAL DE PROCESAMIENTO DE DATOS | | | |
| Profesional Universitario | 9 | 3020 | 06 |
| Profesional Universitario | 2 | 3020 | 03 |
| Dependencia Denominación del cargo | | | |
| DIRECCION GENERAL DE PROCESAMIENTO DE DATOS | | | |
| Profesional Universitario | 9 | 3020 | 06 |
| Profesional Universitario | 2 | 3020 | 03 |

**Artículo 2°** Créanse en la planta de personal del Departamento Administrativo Nacional de Estadística, los siguientes cargos así:

| | | | |
| --- | --- | --- | --- |
| Dependencia Denominación del cargo | N° de cargo | Código | Grado |
| DIRECCION GENERAL DE PROCESAMIENTO DE DATOS | | | |
| Profesional Especializados | 6 | 3010 | 08 |
| Profesional Universitario | 5 | 3020 | 04 |
| Dependencia Denominación del cargo | | | |
| DIRECCION GENERAL DE ANALISIS SOCIO-ECONOMICO | | | |
| Profesional Especializado | 7 | 3010 | 08 |
| Profesional Universitario | 4 | 3020 | 04 |

**Artículo 3°** El presente Decreto rige a partir de la fecha de su expedición.

Comuniquese y cúmplase.

Dado en Bogotá, D. E., a 28 de diciembre de 1978.
