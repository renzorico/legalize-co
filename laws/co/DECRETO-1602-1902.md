# DECRETO 1602 DE 1902

(octubre 29)

### Por el cual se crean unas plazas en el Ramo de telégrafos

**Subtipo:** DECRETO ORDINARIO

## El Vicepresidente de la República, encargado del Poder Ejecutivo,

### En uso de sus facultades legales,

### DECRETA:

**Artículo único.** Créanse las plazas de Inspector seccional de Telégrafos para la reconstrucción y conservación de las líneas telegráficas de Villeta á La Palma y Caparrapí, con la asignación mensual de mil pesos ($ 1,000), y

Cuatro Guardas, así:

| | | |
| --- | --- | --- |
| De Villeta á Qaebradanegra | $ | 150 |
| De Quebradanegra á Ütioa | | 150 |
| De Utica á La Palma | | 150 |
| De Útica á Caparrapí | | 150 |

Comuníquese y publíquese.

Dado en Bogotá, á 29 de Octubre de 1902.

### JOSE MANUEL MARROQUÍN

El Subsecretario de Gobierno, encargado del Despacho,

### Antonio Gutiérrez Rubio
