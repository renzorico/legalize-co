# DECRETO 1564 DE 1931

(septiembre 08)

### Por el cual se fijan los precios de venta de la sal terrestre y la sal marina

**Subtipo:** DECRETO ORDINARIO

## El Presidente de la República de Colombia,

en ejercicio de sus facultades legales, y especialmente de las que le confiere el inciso 3° del artículo 7° de la Ley 78 de 1930,

DECRETA:

**Artículo 1º** Fíjanse los siguientes precios para las materias primas que vende el Estado en las Salinas de Cundinamarca:

| | | | | | | --- | --- | --- | --- | | *Zipaquirá y Nemocón.* | | | | | Vijúa de primera clase, cada 12½ kilogramos | $ | 0 | 74 | | Vijúa de segunda clase, cada 12½ kilogramos | | 0 | 66 | | Agua salada de 20° a 25°, el decalitro | | 0 | 17 | | *Sesquilé.* | | | | | Agua salada de 20° a 25°, el decalitro | $ | 0 | 17 | | *Tansa.* | | | | | Agua salada de 12° a 18°, el decalitro | | 0 | 11 | | *Gacheta.* | | | | | Agua salada de 16° a 22°, el decalitro | | 0 | 13 | |

**Artículo 2º** Los precios fijados en el artículo anterior regirán desde el día nueve del presente mes, con excepción del precio fijado para el agua salada de Gachetá, el cual empezará a regir el día once del mes en curso.

**Artículo 3º** En las Salinas de Chámeza y Recetor, Chita y Muneque, Cumaral y Upín, los precios de venta serán los mismos que rigen actualmente.

**Artículo 4º** A partir del día quince del presente mes los precios de venta de la sal marina, en los siguientes expendios oficiales, serán los que se indican al frente de cada uno de ellos, para sacos de 62½ kilogramos, inclusive el empaque:

| Cartagena | | 4 20 | 3 50 | 3 | |
| Santa Marta | | 4 20 | 3 50 | 3 | |
| Buenaventura | | 5 20 | 4 30 | 3 | 80 |
| Tumaco | | 5 20 | 4 30 | 3 | 80 |

**Artículo 5º** En los demás expendios oficiales de sal marina continuarán rigiendo los precios actuales.

Comuníquese y publíquese.

Dado en Bogotá a 8 de septiembre de 1931.
