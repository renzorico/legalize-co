# DECRETO 1562 DE 1969

(septiembre 22)

### Por el cual se abre un crédito adicional al Presupuesto de Rentas e Ingresos y Ley de Apropiaciones para la vigencia fiscal de 1969 (Ministerio de Relaciones Exteriores), por $ 652.000.00

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales, y

CONSIDERANDO:

Que por medio del numeral 3° del artículo 55 de la Ley 35 de 1968, se autorizó al Gobierno Nacional para abrir, por Decreto, a partir del 1° de enero de 1969, los créditos adicionales al Presupuesto y efectuar los traslados dentro de las apropiaciones para gastos de funcionamiento e inversión, que requiera la buena marcha de la Administración Pública, el cumplimiento del programa de inversiones, los compromisos contractuales con los organismos internacionales de crédito y la Reforma Administrativa;

Que el señor Ministro de Relaciones Exteriores, ha solicitado la apertura de un crédito adicional al Presupuesto de Gastos para la, vigencia fiscal de 1969 (artículo 1166), con el objeto de incrementar la partida de gastos de representación del Servicio Diplomático, que resultó insuficiente;

Que de conformidad con lo dispuesto en el artículo 83 del Decreto-ley 1675 de 1964, el Contralor General de la República expidió el Certificado de Disponibilidad número 12 de 1969, por $ 308.619.188.89, provenientes del superávit fiscal de 1968, del cual se utilizan $ 652.000.00 para financiar el crédito presupuestal de que trata este Decreto, y

Que se han cumplido los requisitos legales para la tramitación de esta clase de negocios,

### DECRETA:

**Artículo 1°** Adiciónanse los cómputos del Presupuesto de Rentas e Ingresos del Tesoro de la Nación para la vigencia fiscal de 1969, con la cantidad de seiscientos cincuenta y dos mil pesos ($ 652.000.00) moneda legal, que, con base en el Certificado de Disponibilidad número 12 del mismo año, expedido por el Contralor General de la República, se incorporarán con imputación al siguiente numeral:

| | |
| --- | --- |
| Presupuesto de Rentas e Ingresos | |
| CAPITULO XV | |
| **Recurso del Balance del Tesoro** | |
| Numeral 165. Superávit fiscal de 1968 | 652.000 |
| Suma el recurso | 652.000 |

**Artículo 2°** Con base en el recurso de que trata el artículo anterior, ábrese el siguiente crédito adicional al Presupuesto de Gastos para la vigencia fiscal de 1969:

| | |
| --- | --- |
| Presupuesto de funcionamiento | |
| **Ministerio de Relaciones Exteriores** | |
| CAPITULO 116 | |
| **Servicio Exterior** | |
| Programa 123 | |
| Servicio Diplomático | |
| Servicios personales | |
| 101 112 105. Artículo 1166. Gastos de representación | 652.000 |
| Suma el crédito | 652.000 |

**Artículo 3°** El présente Decreto rige desde la fecha de su expedición.

Comuníquese y publíquese.

Dado en Bogotá D. E., a 22 de septiembre de 1969.
