# DECRETO 1047 DE 1915

(junio 17)

### Por el cual se organiza la Administración Principal de Correos Nacionales de Tunja

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en ejercicio de sus atribuciones legales,

### decreta:

Organízase la Administración Principal, de Correos Nacionales de Tunja, con el siguiente personal y asignaciones mensuales:

| | | |
| --- | --- | --- |
| Un Administrador Principal, con | $ | 100 |
| Un Tenedor de Libros, con | | 50 |
| Un Oficial de Recibo, con | | 35 |
| Un Oficial de Despacho, con | | 35 |
| Un Oficial de Lista y Apartados, con | | 30 |
| Un Oficial de Encomiendas del interior y del Exterior, con | | 40 |
| Un Oficial de valores, recomendados y especies postales, con | | 35 |
| Un Oficial de Giros Postales y servicio oficial, con | | 40 |
| Un Portero Copista, con | | 25 |
| Un Guarda, con | | 20 |

Para gastos de útiles de escritorio30

Comuníquese y publíquese.

Dado en Bogotá, a 17 de junio de 1915.

### JOSE VICENTE CONCHA

El Ministro de Gobierno,

### Miguel Abadía Méndez.
