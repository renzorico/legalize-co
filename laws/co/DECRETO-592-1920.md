# DECRETO 592 DE 1920

(marzo 12)

### Por el cual se reorganizan unas oficinas en el ramo Telegráfico

**Subtipo:** DECRETO ORDINARIO

### El Presidente de la República de Colombia,

en uso de sus facultades legales,

### decreta:

Reorganízanse las Oficinas Telegráficas de Roldanillo y Ríofrio (Departamento del Valle), cada una con el personal y asignaciones mensuales siguientes:

| | | |
| --- | --- | --- |
| **Roldanillo** | | |
| Un Telegrafista Jefe, con | | 55 |
| Tres Ayudantes, cada uno a | $ 45 | 135 |
| Un Cartero | | 8 |
| Para gastos de alumbrado | | 4 |

| | |
| --- | --- |
| **Riofrio.** | |
| Un Telegrafista Administrador de Correos | 40 |
| Un Cartero | 5 |
| Para gastos de alumbrado | 1 50 |

El presente Decreto empezará a regir diez días después de su fecha.

Comuníquese y publíquese.

Dado en Bogotá a 12 de marzo de 1920.

Por delegación del Excelentísimo señor Presidente de la República, el Ministro de Gobierno, **Luis CUERVO MARQUEZ.**
